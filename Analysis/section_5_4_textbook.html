— 引言 —

第 5.3 节完成了作为 的构造——我们有了加法、减法、乘法、除法和所有代数定律。 现在我们来研究上的序结构（order structure）。

我们知道每个是正数、负数或零。我们希望也是如此。 但由于是 的形式极限， 如果简单地定义"所有项都为正的序列的极限为正"，会遇到问题—— 例如序列 \(0.1, 0.01, 0.001, \dots\) 的所有项都为正，但其极限为 \(0\)。

📖 补充说明：从序回到序列

为了给定义序，我们需要一种更精细的方法， 不能仅仅要求序列的所有项为正。关键在于使用"远离零"的概念—— 如果一个序列的项一致地、永久地保持在一个正的下界之上，则其形式极限为正。

5.4.1：正向/负向远离零的序列

5.4.1（正向/负向远离零的序列）。 设 \((a_n)_{n=1}^\infty\) 是一个。
  • 我们说该序列正向远离零（positively bounded away from zero）， 如果存在正有理数 \(c > 0\) 使得对所有 \(n \geq 1\) 有 \(a_n \geq c\)（特别地，序列完全为正）；
  • 我们说该序列负向远离零（negatively bounded away from zero）， 如果存在负有理数 \(-c < 0\) 使得对所有 \(n \geq 1\) 有 \(a_n \leq -c\)（特别地，序列完全为负）。
例 5.4.2。
  • 序列 \(1.1, 1.01, 1.001, 1.0001, \dots\) 正向远离零（所有项 \(\geq 1\)）；
  • 序列 \(-1.1, -1.01, -1.001, -1.0001, \dots\) 负向远离零；
  • 序列 \(1, -1, 1, -1, \dots\) 远离零但既非正向也非负向远离零。
📖 补充说明：Lean 中的定义

在 Section_5_4.lean 中：

abbrev BoundedAwayPos (a:ℕ → ℚ) : Prop := ∃ (c:ℚ), c > 0 ∧ ∀ n, a n ≥ c abbrev BoundedAwayNeg (a:ℕ → ℚ) : Prop := ∃ (c:ℚ), c > 0 ∧ ∀ n, a n ≤ -c

5.4.3：正实数与负实数

5.4.3（正实数与负实数）。
  • 一个 \(x\) 被称为正的（positive）， 如果它可以写成 \(x = \operatorname{LIM}_{n\to\infty} a_n\)，其中 \((a_n)\) 是正向远离零的 ；
  • \(x\) 被称为负的（negative）， 如果它可以写成 \(x = \operatorname{LIM}_{n\to\infty} a_n\)，其中 \((a_n)\) 是负向远离零的 。
📖 补充说明：Lean 中的 IsPos / IsNeg
abbrev Real.IsPos (x:Real) : Prop := ∃ a:ℕ → ℚ, BoundedAwayPos a ∧ (a:Sequence).IsCauchy ∧ x = LIM a abbrev Real.IsNeg (x:Real) : Prop := ∃ a:ℕ → ℚ, BoundedAwayNeg a ∧ (a:Sequence).IsCauchy ∧ x = LIM a

5.4.4：正实数的基本性质

5.4.4（正实数的基本性质）。 对每个实数 \(x\)，以下三个陈述恰好有一个为真：
  1. \(x = 0\)；
  2. \(x\) 是正的；
  3. \(x\) 是负的。
此外：
  • \(x\) 为负当且仅当 \(-x\) 为正；
  • 如果 \(x\) 和 \(y\) 为正，则 \(x + y\) 和 \(xy\) 也为正。
证明见习题 5.4.1。
📖 补充说明：三分法与 Lean 形式化

命题 5.4.4 建立了实数的三分法（trichotomy）： 每个实数要么为零、要么为正、要么为负，且三者互斥。 这是后续所有序性质的基础。

Lean 中 Real.trichotomous 定理的证明（习题 5.4.1）是本节形式化的核心：

theorem Real.trichotomous (x:Real) : x = 0 ∨ x.IsPos ∨ x.IsNeg := by sorry

该证明的关键洞见是：如果 \(x \neq 0\)，那么选定的 Cauchy 序列最终必须 与零序列保持一个固定的距离，从而可以被调整为正向或负向远离零。

5.4.5：绝对值

5.4.5（绝对值）。 设 \(x\) 是一个。
  • 如果 \(x\) 是正的，则 \(|x| = x\)；
  • 如果 \(x\) 是负的，则 \(|x| = -x\)；
  • 如果 \(x = 0\)，则 \(|x| = 0\)。

5.4.6：实数的序

5.4.6（实数的序）。 设 \(x\) 和 \(y\) 是。
  • \(x > y\) 当且仅当 \(x - y\) 是正实数；
  • \(x < y\) 当且仅当 \(x - y\) 是负实数；
  • \(x \geq y\) 当且仅当 \(x > y\) 或 \(x = y\)；
  • \(x \leq y\) 当且仅当 \(x < y\) 或 \(x = y\)。
注。 该定义与上的序定义（定义 4.2.8）一致。 即，如果两个 \(q, q'\) 在中满足 \(q < q'\)， 那么它们在中也满足 \(q < q'\)。绝对值的定义也同样一致。

5.4.7：序的性质

5.4.7。 命题 4.2.9 中所有对成立的序性质对也成立。 特别地：
  1. （三分法）对任意 \(x, y\)，\(x > y\)、\(x < y\)、\(x = y\) 恰好一个成立；
  2. （反对称性）\(x < y\) 当且仅当 \(y > x\)；
  3. （传递性）如果 \(x < y\) 且 \(y < z\)，则 \(x < z\)；
  4. （加法保序）如果 \(x < y\)，则 \(x + z < y + z\)；
  5. （正数乘法保序）如果 \(x < y\) 且 \(z\) 为正，则 \(xz < yz\)。
我们只证明 (e) 并留其余给习题 5.4.2。由于 \(x < y\)，\(y - x\) 为正。 由命题 5.4.4，\((y - x)z = yz - xz\) 为正，因此 \(xz < yz\)。
📖 补充说明：Lean 中的序结构

在 Lean 中，序关系的定义直接对应于教科书：

instance Real.instLT : LT Real where lt x y := (x-y).IsNeg instance Real.instLE : LE Real where le x y := (x < y) ∨ (x = y)

最终通过 Real.instLinearOrder 将实数注册为一个线性序。 虽然非可计算的（需要经典逻辑），但它提供了所有我们所期待的序推理工具。

5.4.8：正实数的倒数

5.4.8。 设 \(x\) 为正实数。则 \(x^{-1}\) 也为正。此外，如果 \(y\) 也是正数且 \(x > y\)， 则 \(x^{-1} < y^{-1}\)。
由于 \(xx^{-1} = 1\)，\(x^{-1}\) 不能为零（因为 \(x \cdot 0 = 0 \neq 1\)）。 由命题 5.4.4，正数乘以负数得负数；因此 \(x^{-1}\) 不能为负（那样会导致 \(xx^{-1} = 1\) 为负，矛盾）。由三分法，\(x^{-1}\) 必为正。 现设 \(y\) 也为正，则 \(x^{-1}\) 和 \(y^{-1}\) 都为正。假设 \(x > y\)。 如果 \(x^{-1} \geq y^{-1}\)，则由命题 5.4.7 得 \(xx^{-1} > yx^{-1} \geq yy^{-1}\)， 即 \(1 > 1\)，矛盾。因此必有 \(x^{-1} < y^{-1}\)。

5.4.9：非负序列的形式极限非负

5.4.9。 设 \(a_1, a_2, a_3, \dots\) 是的非负 。 则 \(\operatorname{LIM}_{n\to\infty} a_n\) 是一个非负实数。
用反证法。假设 \(x := \operatorname{LIM} a_n\) 为负。 则存在负向远离零的序列 \(b_n\) 使得 \(x = \operatorname{LIM} b_n\)， 即存在负有理数 \(-c < 0\) 使得对所有 \(n\) 有 \(b_n \leq -c\)。 而由假设 \(a_n \geq 0\)。因此 \(a_n\) 和 \(b_n\) 永远不可能是 \(c/2\)-接近的 （因为 \(|a_n - b_n| \geq c > c/2\)），故它们不是最终 \(c/2\)-接近的， 从而不等价。但这与两者都以 \(x\) 为形式极限矛盾。

5.4.10

5.4.10。 设 \((a_n)\) 和 \((b_n)\) 是 ， 且对所有 \(n\) 有 \(a_n \geq b_n\)。则 \(\operatorname{LIM} a_n \geq \operatorname{LIM} b_n\)。
将命题 5.4.9 应用于序列 \(a_n - b_n\)。

注 5.4.11

注 5.4.11。 推论 5.4.10 中的 \(\geq\) 不能替换为 \(>\)： 若 \(a_n := 1 + 1/n\)，\(b_n := 1 - 1/n\)，则对所有 \(n\) 有 \(a_n > b_n\)， 但 \(\operatorname{LIM} a_n = \operatorname{LIM} b_n = 1\)，并非严格大于。

5.4.12：有理数界

5.4.12（用有理数界定实数）。 设 \(x\) 为正实数。则存在正有理数 \(q\) 使得 \(q \leq x\)，且存在正整数 \(N\) 使得 \(x \leq N\)。
由于 \(x\) 为正，它是某个正向远离零的 \((a_n)\) 的形式极限。 由引理 5.1.15，该序列有界。故存在有理数 \(q > 0\) 和 \(r\) 使得 \(q \leq a_n \leq r\) 对所有 \(n\) 成立。 由命题 4.4.1，存在整数 \(N\) 使得 \(r \leq N\)。 由推论 5.4.10 得 \(q \leq x \leq N\)。

5.4.13：阿基米德性质

5.4.13（阿基米德性质）。 设 \(x\) 为实数，\(\varepsilon\) 为正实数。则存在正整数 \(M\) 使得 \(M\varepsilon > x\)。
若 \(x \leq 0\)，取 \(M = 1\) 即可。若 \(x > 0\)，则 \(x/\varepsilon\) 为正， 由命题 5.4.12 存在正整数 \(N\) 使得 \(x/\varepsilon \leq N\)。 令 \(M := N + 1\)，则 \(x/\varepsilon < M\)，乘以 \(\varepsilon\) 即得。
📖 补充说明：阿基米德性质的意义

阿基米德性质说：不管 \(x\) 有多大、\(\varepsilon\) 有多小， 只要把 \(\varepsilon\) 重复加给自己足够多次，最终一定会超过 \(x\)。 这看似显然，但并非所有有序域都满足此性质（例如超实数就不满足）。 它是实数系区别于其他有序域的关键特征之一。

5.4.14：有理数在实数中稠密

5.4.14。 对任意两个实数 \(x < y\)，存在有理数 \(q\) 使得 \(x < q < y\)。
证明见习题 5.4.5。
📖 补充说明：有理数的稠密性

命题 5.4.14 说有理数在实数中是稠密（dense）的： 任意两个不同的实数之间都存在一个有理数。 这是分析学中一个极其重要的事实——它意味着任何实数都可以用有理数任意逼近。

在 Lean 中，该命题对应 Real.rat_between，其证明依赖于 阿基米德性质（推论 5.4.13）和一些巧妙的估计。

— 第 5.4 节总结 —

第 5.4 节为实数建立了完整的序结构。关键步骤是：

  1. 引入"正向/负向远离零"的概念来定义正实数和负实数（定义 5.4.1–5.4.3）；
  2. 证明三分法及相关性质（命题 5.4.4，习题 5.4.1）；
  3. 定义绝对值和序关系（定义 5.4.5–5.4.6）；
  4. 验证序的代数性质（命题 5.4.7，习题 5.4.2）；
  5. 证明阿基米德性质和有理数的稠密性（推论 5.4.13，命题 5.4.14）。

至此，实数已具有完整的域结构和序结构——它们构成一个有序域（ordered field）。 在下一节中，我们将探索实数相对于有理数的核心优势：上确界性质（least upper bound property）。

📖 补充说明：Lean 中的线性序实例

尽管 IsPos 和 IsNeg 不可判定（需要经典逻辑）， Lean 仍然可以通过 Real.instLinearOrder 将实数注册为线性序：

noncomputable instance Real.instLinearOrder : LinearOrder Real where le_refl := ... le_trans := ... lt_iff_le_not_ge := ... le_antisymm := ... le_total := ... toDecidableLE := Classical.decRel _

注意 toDecidableLE := Classical.decRel _ 这行： 它使用经典逻辑的排中律来"假装"序关系是可判定的。 这就是为什么实数上的 ≤ 不可计算——但我们可以照常推理。
