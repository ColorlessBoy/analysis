— 引言 —

在前几节中，我们讨论了序列的极限——序列"趋向"某个特定值的行为。 但是，有些序列虽然不收敛，却有某种"想成为极限"的值。

例如，考虑序列 \[ 1.1,\ -1.01,\ 1.001,\ -1.0001,\ \ldots \] 该序列不收敛（奇数项趋向 \(1\)，偶数项趋向 \(-1\)）， 但从某种意义上说，\(1\) 和 \(-1\) 都是序列的"极限点"——序列的项无限频繁地接近这两个值。

本节的目标就是严格化这一直觉，引入极限点（limit point）、上极限（limsup）和下极限（liminf）的概念， 并建立它们与极限的深刻联系，最终推导出实数完备性的一个重要版本：Cauchy 序列收敛定理。

6.4.1：极限点

6.4.1（极限点）。 设 \((a_n)_{n=m}^\infty\) 是实数序列，\(x\) 是实数，\(\varepsilon > 0\)。称 \(x\) 是序列的 ，如果存在 \(n \geq m\) 使得 \(|a_n - x| \leq \varepsilon\)。 称 \(x\) 是序列， 如果对每个 \(N \geq m\) 都存在 \(n \geq N\) 使得 \(|a_n - x| \leq \varepsilon\)。 称 \(x\) 是序列的（或聚点）， 如果对每个 \(\varepsilon > 0\)，\(x\) 都是序列连续 \(\varepsilon\)-附着的。
注 6.4.2。 换言之，\(x\) 是极限点当且仅当：对所有 \(\varepsilon > 0\) 和所有 \(N \geq m\)， 存在 \(n \geq N\) 使得 \(|a_n - x| \leq \varepsilon\)。 这与极限的定义几乎相同，区别在于：极限要求"对充分大的所有 \(n\)"， 而极限点只要求"对充分大的无穷多个 \(n\)"。

6.4.3–6.4.4

6.4.3。 设 \(a_n := 1 - 10^{-n}\)，即序列 \(0.9, 0.99, 0.999, \ldots\)。 则 \(1\) 是该序列的极限点。实际上，该序列收敛到 \(1\)，而收敛的极限也是极限点（见命题 6.4.5）。
6.4.4。 设序列为 \(1.1,\ -1.01,\ 1.001,\ -1.0001,\ \ldots\)， 其中奇数项为 \(1 + 10^{-n}\)，偶数项为 \(-(1 + 10^{-n})\)（从 \(n=1\) 开始）。 则 \(1\) 是该序列的极限点：对任意 \(\varepsilon > 0\) 和 \(N\)，只需取充分大的奇数 \(n \geq N\)， 就有 \(|a_n - 1| = 10^{-n} \leq \varepsilon\)。 类似地，\(-1\) 也是该序列的极限点。 该序列不收敛，但有两个极限点 \(1\) 和 \(-1\)。

6.4.5：极限是极限点

6.4.5（极限是极限点）。 设 \((a_n)_{n=m}^\infty\) 是收敛到极限 \(L\) 的实数序列。则 \(L\) 是该序列的极限点， 并且是唯一的极限点。
证明.

首先证明 \(L\) 是极限点。设 \(\varepsilon > 0\) 和 \(N \geq m\)。 由于序列收敛到 \(L\)，存在 \(N' \geq m\) 使得对所有 \(n \geq N'\) 有 \(|a_n - L| \leq \varepsilon\)。 取 \(n = \max(N, N')\)，则 \(n \geq N\) 且 \(|a_n - L| \leq \varepsilon\)。所以 \(L\) 是极限点。

再证唯一性。设 \(x\) 也是极限点，任取 \(\varepsilon > 0\)。 由收敛性，存在 \(N\) 使得对所有 \(n \geq N\) 有 \(|a_n - L| \leq \varepsilon/2\)。 由于 \(x\) 是极限点，存在 \(n \geq N\) 使 \(|a_n - x| \leq \varepsilon/2\)。 于是 \(|x - L| \leq |x - a_n| + |a_n - L| \leq \varepsilon\)。 由 \(\varepsilon\) 的任意性，\(x = L\)。

∎

6.4.6：上极限与下极限

6.4.6（上极限与下极限）。 设 \((a_n)_{n=m}^\infty\) 是实数序列。对每个 \(N \geq m\)，定义 \[ a_N^+ := \sup(a_n)_{n=N}^\infty, \quad a_N^- := \inf(a_n)_{n=N}^\infty. \] 则序列 \((a_N^+)_{N=m}^\infty\) 递减，\((a_N^-)_{N=m}^\infty\) 递增。 定义序列 \((a_n)_{n=m}^\infty\) 的为 \[ \limsup_{n\to\infty} a_n := \inf_{N \geq m} a_N^+ = \inf_{N \geq m} \sup_{n \geq N} a_n, \] 定义为 \[ \liminf_{n\to\infty} a_n := \sup_{N \geq m} a_N^- = \sup_{N \geq m} \inf_{n \geq N} a_n. \] 这两个值都是拓展实数（可以为 \(\pm\infty\)）。

6.4.7–6.4.10

6.4.7。 设 \(a_n := (-1)^n\)，序列为 \(-1, 1, -1, 1, \ldots\)。 对每个 \(N\)，集合 \(\{a_n : n \geq N\}\) 均为 \(\{-1, 1\}\)， 故 \(a_N^+ = 1\)，\(a_N^- = -1\)。 因此 \(\limsup_{n\to\infty} (-1)^n = 1\)，\(\liminf_{n\to\infty} (-1)^n = -1\)。
6.4.8。 设 \(a_n := 1/n\)（\(n \geq 1\)）。 则 \(a_N^+ = \sup(1/n)_{n \geq N} = 1/N\)（最大值在 \(n=N\) 取到）， \(a_N^- = \inf(1/n)_{n \geq N} = 0\)（下确界不可达）。 因此 \(\limsup_{n\to\infty} 1/n = \inf_N (1/N) = 0\)， \(\liminf_{n\to\infty} 1/n = \sup_N 0 = 0\)。 两者相等，且等于极限 0。
6.4.9。 设 \(a_n := n\)（递增无界序列）。 则 \(a_N^+ = +\infty\)，\(a_N^- = N\)。 故 \(\limsup_{n\to\infty} n = +\infty\)，\(\liminf_{n\to\infty} n = \sup_N N = +\infty\)。
6.4.10。 设 \(a_n := -n\)（递减无界序列）。 则 \(a_N^+ = -N\)，\(a_N^- = -\infty\)。 故 \(\limsup_{n\to\infty} (-n) = \inf_N (-N) = -\infty\)， \(\liminf_{n\to\infty} (-n) = -\infty\)。

6.4.12：上下极限的性质

6.4.12（上下极限的性质）。 设 \((a_n)_{n=m}^\infty\) 是实数序列，令 \(L^+ := \limsup_{n\to\infty} a_n\)， \(L^- := \liminf_{n\to\infty} a_n\)。
  • (a) 对任意 \(x > L^+\)，存在 \(N \geq m\) 使得对所有 \(n \geq N\) 有 \(a_n \leq x\)。 对任意 \(x < L^+\)，对所有 \(N \geq m\) 存在 \(n \geq N\) 使 \(a_n > x\)。
  • (b) 对任意 \(x < L^-\)，存在 \(N \geq m\) 使得对所有 \(n \geq N\) 有 \(a_n \geq x\)。 对任意 \(x > L^-\)，对所有 \(N \geq m\) 存在 \(n \geq N\) 使 \(a_n < x\)。
  • (c) \(\inf(a_n)_{n=m}^\infty \leq L^- \leq L^+ \leq \sup(a_n)_{n=m}^\infty\)。
  • (d) 若 \(x\) 是 \((a_n)_{n=m}^\infty\) 的极限点，则 \(L^- \leq x \leq L^+\)。
  • (e) 若 \(L^+\)（或 \(L^-\)）是有限实数，则它本身是序列的极限点。
  • (f) 序列 \((a_n)_{n=m}^\infty\) 收敛到 \(L\) 当且仅当 \(L^+ = L^- = L\)。
证明（要点）。

(a) 若 \(x > L^+ = \inf_N a_N^+\)，则存在 \(N\) 使 \(a_N^+ \leq x\)，即 \(\sup_{n \geq N} a_n \leq x\)， 故对所有 \(n \geq N\) 有 \(a_n \leq x\)。 若 \(x < L^+\)，则对所有 \(N\) 有 \(a_N^+ \geq L^+ > x\)， 故存在 \(n \geq N\) 使 \(a_n > x\)。

(c) 对所有 \(N\)，有 \(\inf_{n \geq m} a_n \leq \inf_{n \geq N} a_n \leq \sup_{n \geq N} a_n \leq \sup_{n \geq m} a_n\)， 取关于 \(N\) 的上确界或下确界即得 \(\inf \leq L^- \leq L^+ \leq \sup\)。

(f) 若收敛到 \(L\)，对 \(\varepsilon > 0\) 存在 \(N\) 使所有 \(n \geq N\) 满足 \(|a_n - L| \leq \varepsilon\)， 因此 \(L^- \leq L^+ \leq a_N^+ \leq L + \varepsilon\) 且 \(L^- \geq a_N^- \geq L - \varepsilon\)， 由 \(\varepsilon\) 任意性得 \(L^- = L^+ = L\)。 反之若 \(L^- = L^+ = L\)，对任意 \(\varepsilon > 0\)，由 (a)(b) 存在 \(N\) 使所有 \(n \geq N\) 有 \(|a_n - L| \leq \varepsilon\)。

∎

6.4.13：比较原理

6.4.13（比较原理）。 设 \((a_n)_{n=m}^\infty\) 和 \((b_n)_{n=m}^\infty\) 是实数序列，对所有 \(n \geq m\) 有 \(a_n \leq b_n\)。则
  • \(\sup(a_n)_{n=m}^\infty \leq \sup(b_n)_{n=m}^\infty\)；
  • \(\inf(a_n)_{n=m}^\infty \leq \inf(b_n)_{n=m}^\infty\)；
  • \(\limsup_{n\to\infty} a_n \leq \limsup_{n\to\infty} b_n\)；
  • \(\liminf_{n\to\infty} a_n \leq \liminf_{n\to\infty} b_n\)。
证明.

由 \(a_n \leq b_n\) 知 \(\{a_n : n \geq m\} \subseteq \{x : x \leq b_n \text{ 对某 } n\}\)， 故 \(\sup(a_n) \leq \sup(b_n)\)。对 \(\inf\) 类似。 对于 limsup，有 \(\sup_{n \geq N} a_n \leq \sup_{n \geq N} b_n\)， 取关于 \(N\) 的下确界即得 \(\limsup a_n \leq \limsup b_n\)。liminf 类似。

∎
补充说明：Lean 中的对应定理
  • sup_mono：\(a_n \leq b_n \Rightarrow \sup(a) \leq \sup(b)\)
  • inf_mono：\(a_n \leq b_n \Rightarrow \inf(a) \leq \inf(b)\)
  • limsup_mono：\(a_n \leq b_n \Rightarrow \limsup(a) \leq \limsup(b)\)
  • liminf_mono：\(a_n \leq b_n \Rightarrow \liminf(a) \leq \liminf(b)\)

6.4.14：夹逼定理

6.4.14（夹逼定理）。 设 \((a_n)_{n=m}^\infty\)、\((b_n)_{n=m}^\infty\)、\((c_n)_{n=m}^\infty\) 是实数序列， 对所有 \(n \geq m\) 有 \(a_n \leq b_n \leq c_n\)。 若 \(\lim_{n\to\infty} a_n = \lim_{n\to\infty} c_n = L\)，则 \((b_n)_{n=m}^\infty\) 也收敛，且 \(\lim_{n\to\infty} b_n = L\)。
证明.

由命题 6.4.12(f)，\(\limsup a_n = \liminf a_n = L\) 且 \(\limsup c_n = \liminf c_n = L\)。 由引理 6.4.13，\(L = \limsup a_n \leq \limsup b_n\) 且 \(\limsup b_n \leq \limsup c_n = L\)， 故 \(\limsup b_n = L\)。类似地 \(\liminf b_n = L\)。再由 6.4.12(f) 得 \(b_n \to L\)。

∎

6.4.17：零判别法

6.4.17（零判别法）。 设 \((a_n)_{n=m}^\infty\) 是实数序列。则 \[ \lim_{n\to\infty} a_n = 0 \iff \lim_{n\to\infty} |a_n| = 0. \]
证明.

注意 \(-|a_n| \leq a_n \leq |a_n|\)。 若 \(|a_n| \to 0\)，则 \(-|a_n| \to 0\)，由夹逼定理得 \(a_n \to 0\)。 反之若 \(a_n \to 0\)，则 \(\limsup |a_n| \leq \limsup a_n = 0\) 且 \(\liminf |a_n| \geq \liminf a_n = 0\)，故 \(|a_n| \to 0\)。 （也可用 \(\limsup |a_n| \leq \limsup a_n\) 和 \(\limsup |a_n| \leq \limsup (-a_n)\) 推导。）

∎

6.4.18：实数的完备性

6.4.18（实数的完备性）。 实数序列 \((a_n)_{n=1}^\infty\) 是 Cauchy 序列当且仅当它收敛。
证明.

（收敛 ⟹ Cauchy） 已在命题 6.1.12 中证明：若序列收敛到 \(L\)，则对任意 \(\varepsilon > 0\)，存在 \(N\) 使得对所有 \(n, m \geq N\) 有 \(|a_n - a_m| \leq |a_n - L| + |L - a_m| \leq \varepsilon/2 + \varepsilon/2 = \varepsilon\)。

（Cauchy ⟹ 收敛） 设 \((a_n)_{n=1}^\infty\) 是 Cauchy 序列。由推论 6.1.17，序列有界，故存在实数 \(M\) 使所有 \(a_n \in [-M, M]\)。 因此 \(-\infty < L^- \leq L^+ < +\infty\)（均为有限实数）。

若 \(L^- < L^+\)，取 \(\varepsilon = (L^+ - L^-)/3 > 0\)。 由命题 6.4.12(a)，存在无穷多 \(n\) 使 \(a_n > L^+ - \varepsilon = L^- + 2\varepsilon\)； 由命题 6.4.12(b)，也存在无穷多 \(n\) 使 \(a_n < L^- + \varepsilon\)。 这意味着序列无穷次振幅大于 \(\varepsilon\)，与 Cauchy 性矛盾。 故 \(L^- = L^+\)，由命题 6.4.12(f) 序列收敛。

∎
补充说明：Lean 中的对应定理
-- Theorem 6.4.18: Cauchy iff convergent theorem Cauchy_iff_convergent {a : Sequence} : a.IsCauchy ↔ a.Convergent

这是实数完备性的核心表现。在有理数中，Cauchy 序列不一定收敛（例如逼近 \(\sqrt{2}\) 的有理序列）； 而实数正是通过补全有理数中所有 Cauchy 序列的极限而构造的，因此在实数中 Cauchy 等价于收敛。

个人感悟：limsup/liminf 的深刻意义

上极限和下极限是分析学中最优雅的概念之一：它们对任意有界序列都有定义， 不像极限那样要求序列收敛。从某种意义上说，limsup 和 liminf 是极限的"最坏情形"和"最好情形"的版本—— limsup 捕捉序列无穷次能达到的最高位置，liminf 捕捉最低位置。

命题 6.4.12(f) 表明，序列收敛当且仅当这两个"极端极限"恰好相同。 这种思路在数学分析中非常有用：要证明极限存在，只需分别估计 limsup 和 liminf 的上界和下界， 用夹逼迫使它们相等。

在 Lean 的形式化过程中，limsup 和 liminf 需要在扩展实数系 \(\mathbb{R}^*\) 中定义（因为可能为 \(\pm\infty\)）， 但命题 6.4.12(f) 的证明要求它们是有限实数。这种"扩展"与"有限"之间的转换需要额外的类型推导。
