常数序列与调和序列

一个特别简单的极限是常数序列 \(c, c, c, c, \ldots\) 的极限；显然有 \[ \lim_{n\to\infty} c = c \] 对任意常数 \(c\) 成立（为什么？）。

此外，在命题 6.1.11 中我们证明了 \(\lim_{n\to\infty} 1/n = 0\)。这立即推出：

补充说明：Lean 中的常数序列
/-- Constant sequences converge to their value -/ theorem Sequence.lim_of_const (c:ℝ) : ((fun (_:ℕ) ↦ c):Sequence).TendsTo c

这一基本事实在后续证明中反复使用（例如 \(x = 1\) 时的几何序列、引理 6.5.3 中 \(x = 1\) 的情形）。

6.5.1：\(1/n^{1/k} \to 0\)

6.5.1。 对每个整数 \(k \geq 1\)，有 \[ \lim_{n\to\infty} \frac{1}{n^{1/k}} = 0. \]
证明.

由引理 5.6.6 知 \(1/n^{1/k}\) 是 \(n\) 的递减函数，且有下界 0。 由命题 6.3.8（递减序列版本），该序列收敛到某个极限 \(L \geq 0\)： \[ L = \lim_{n\to\infty} \frac{1}{n^{1/k}}. \] 将其 \(k\) 次方并利用极限定律（更精确地说，定理 6.1.19(b) 和归纳法），得到 \[ L^k = \lim_{n\to\infty} \frac{1}{n}. \] 由命题 6.1.11 知 \(L^k = 0\)；但这意味着 \(L\) 不能为正（否则 \(L^k\) 为正）， 故 \(L = 0\)，证毕。

∎
补充说明：Lean 中的形式化

推论 6.5.1 在 Section_6_5.lean 中忠实再现了教材的证明结构。Lean 中序列的幂运算通过以下实例定义：

instance Sequence.inst_pow: Pow Sequence ℕ where pow a k := { m := a.m seq n := if n ≥ a.m then a n ^ k else 0 vanish := by grind }

核心定理：

/-- Corollary 6.5.1 -/ theorem Sequence.lim_of_power_decay {k:ℕ} : ((fun (n:ℕ) ↦ 1/((n:ℝ)+1)^(1/(k+1:ℝ))):Sequence).TendsTo 0

证明先用 convergent_of_antitone（递减有下界 → 收敛）， 再对极限的幂使用归纳法（pow_succ + lim_mul）， 最后由 \(L^{k+1} = 0\) 推出 \(L = 0\)。

补充说明：Lean 中的形式化

引理 6.5.2 在 Lean 中被拆分为三个定理，分别对应三种情形：

/-- |x| < 1: x^n → 0 -/ theorem Sequence.lim_of_geometric {x:ℝ} (hx: |x| < 1) : ((fun (n:ℕ) ↦ x^n):Sequence).TendsTo 0 /-- x = 1: x^n → 1 -/ theorem Sequence.lim_of_geometric' {x:ℝ} (hx: x = 1) : ((fun (n:ℕ) ↦ x^n):Sequence).TendsTo 1 /-- x = -1 or |x| > 1: 发散 -/ theorem Sequence.lim_of_geometric'' {x:ℝ} (hx: x = -1 ∨ |x| > 1) : ((fun (n:ℕ) ↦ x^n):Sequence).Divergent

第一种情形的关键工具是 pow_unbounded_of_one_lt：由于 \(1/|x| > 1\)， \((1/|x|)^k\) 可以超过任意界，从而对足够大的 \(n\) 有 \(|x|^n < \varepsilon\)。 第三种情形中 \(|x| > 1\) 时利用收敛序列必有界，而 \(|x|^n\) 无界导出矛盾； \(x = -1\) 时直接证明序列非 Cauchy（相邻项距离为 2）。

补充说明：Lean 中的形式化
/-- Lemma 6.5.3 / Exercise 6.5.3 -/ theorem Sequence.lim_of_roots {x:ℝ} (hx: x > 0) : ((fun (n:ℕ) ↦ x^(1/(n+1:ℝ))):Sequence).TendsTo 1

Lean 证明遵循教材提示的分情况讨论策略。对 \(x \geq 1\) 的情形， 利用 pow_unbounded_of_one_lt：由于 \(1 + \varepsilon > 1\)， 存在 \(k\) 使 \(x < (1+\varepsilon)^k\)，从而当 \(n \geq k\) 时 \(x^{1/(n+1)} \leq 1 + \varepsilon\)。对 \(x < 1\) 的情形， 利用 \(1/x > 1\) 的结果，再取倒数（tendsTo_inv）。

个人感悟："标准极限"为何重要

本节看似只是在罗列几个"显然"的极限，但它们实际上是整个分析学的基石工具。 \(x^n \to 0\)（当 \(|x| < 1\)）是几何级数收敛的核心； \(x^{1/n} \to 1\) 是定义实指数运算（第 6.7 节）的关键引理—— 正是因为 \(x^{1/k}\) 可以任意接近 1，我们才能控制 \(x^{q_n}\) 与 \(x^{q_m}\) 的差距， 从而证明 \(x^{q_n}\) 是 Cauchy 序列。

从形式化的角度看，这些"标准极限"的证明往往需要将数学直觉（"显然趋于 0"） 翻译成精确的 \(\varepsilon\)-\(N\) 论证。例如 \(x^n \to 0\) 的 Lean 证明 依赖于 pow_unbounded_of_one_lt——即阿基米德性质的具体体现： \((1/|x|)^k\) 终将超过任何界。这种"将直觉拆解为公理"的过程， 正是形式化数学的核心价值。
