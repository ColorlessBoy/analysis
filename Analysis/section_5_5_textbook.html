— 引言 —

第 5.4 节表明实数具有与有理数完全相同的序结构：都是阿基米德有序域， 有理数在其中稠密。那么实数的"优越性"究竟体现在哪里？

答案是上确界性质（least upper bound property），也叫完备性（completeness）： 实数 \(\mathbb{R}\) 的每个非空有上界的子集都有一个最小上界（即上确界）。 有理数 \(\mathbb{Q}\) 不具有这一性质——例如集合 \(\{q \in \mathbb{Q} : q^2 < 2\}\) 在 \(\mathbb{Q}\) 中没有上确界，因为 \(\sqrt{2} \notin \mathbb{Q}\)。 正是这个性质使得实数可以"填补"有理数中的所有"间隙"。

📖 补充说明：完备性的不同表述

实数的完备性可以有多种等价的表述方式：

  • 上确界性质： 每个非空有上界的子集都有上确界（本节采用）；
  • Cauchy 完备性： 每个实数 Cauchy 序列都收敛（第 6 章）；
  • 区间套定理： 闭区间套的交集非空；
  • Bolzano-Weierstrass 性质： 有界序列有收敛子列。

这些表述在实数中是等价的，但在更一般的度量空间中可能不等价。

5.5.1：上界

5.5.1（上界）。 设 \(E \subseteq \mathbb{R}\)，\(M \in \mathbb{R}\)。称 \(M\) 为 \(E\) 的一个上界（upper bound）， 如果对每个 \(x \in E\) 都有 \(x \leq M\)。
例 5.5.2–5.5.4。
  • 令 \(E = \{x \in \mathbb{R} : 0 \leq x \leq 1\}\)，则 1 是 \(E\) 的一个上界（事实上每个 \(\geq 1\) 的数都是上界）；
  • \(\mathbb{R}^+ = \{x > 0\}\) 没有上界；
  • 空集 \(\emptyset\) 的每个实数都是上界（空真成立）。
📖 补充说明：Lean 中的上界

Lean 使用 Mathlib 中标准的 upperBounds 定义：

theorem Real.upperBound_def (E: Set Real) (M: Real) : M ∈ upperBounds E ↔ ∀ x ∈ E, x ≤ M := mem_upperBounds

5.5.5：

5.5.5（）。 设 \(E \subseteq \mathbb{R}\)，\(M \in \mathbb{R}\)。称 \(M\) 为 \(E\) 的最小上界或上确界 （least upper bound / supremum），如果：
  1. \(M\) 是 \(E\) 的一个上界；
  2. 对 \(E\) 的任何上界 \(M'\)，有 \(M' \geq M\)。
例 5.5.6–5.5.7。
  • \(E = [0, 1]\) 的上确界是 1（所有其他上界都大于 1）；
  • 空集没有上确界（因为任何实数都是上界，但没有最小的上界）。

5.5.8：上确界的唯一性

5.5.8（上确界的唯一性）。 设 \(E \subseteq \mathbb{R}\)。则 \(E\) 至多有一个最小上界。
设 \(M_1\) 和 \(M_2\) 都是最小上界。由于 \(M_1\) 是最小上界而 \(M_2\) 是上界， 由定义得 \(M_2 \geq M_1\)。同理，\(M_1 \geq M_2\)。故 \(M_1 = M_2\)。

5.5.9：上确界的存在性

5.5.9（最小上界的存在性）。 设 \(E\) 是 \(\mathbb{R}\) 的非空子集。如果 \(E\) 有上界，则 \(E\) 恰好有一个最小上界。
由命题 5.5.8，至多有一个最小上界。我们需要证明至少存在一个。 由于 \(E\) 非空，取 \(x_0 \in E\)。 设 \(n \geq 1\)。已知 \(E\) 有上界 \(M\)。由阿基米德性质（推论 5.4.13）， 存在整数 \(K\) 使得 \(K/n \geq M\)（从而 \(K/n\) 也是上界）。 同样地，存在整数 \(L\) 使得 \(L/n < x_0\)（从而 \(L/n\) 不是上界）。 由于 \(K/n\) 是上界而 \(L/n\) 不是，存在整数 \(L < m_n \leq K\) 使得 \(m_n/n\) 是上界但 \((m_n - 1)/n\) 不是（见习题 5.5.2）。 这定义了整数序列 \(m_1, m_2, m_3, \dots\)。 设 \(N \geq 1\)。对任意 \(n, n' \geq N\)，由于 \(m_n/n\) 是上界而 \((m_{n'} - 1)/n'\) 不是， 我们有 \(m_n/n > (m_{n'} - 1)/n'\)，经代数变形得： \[ \frac{m_n}{n} - \frac{m_{n'}}{n'} > -\frac{1}{n'} \geq -\frac{1}{N}. \] 同理，\(m_{n'}/n' > (m_n - 1)/n\) 给出： \[ \frac{m_n}{n} - \frac{m_{n'}}{n'} \leq \frac{1}{n} \leq \frac{1}{N}. \] 因此对任意 \(n, n' \geq N\)： \[ \left|\frac{m_n}{n} - \frac{m_{n'}}{n'}\right| \leq \frac{1}{N}. \] 故 \(m_n/n\) 是 （习题 5.5.4）。 定义实数： \[ S := \operatorname{LIM}_{n\to\infty} \frac{m_n}{n}. \] 由习题 5.3.5，也有 \(S = \operatorname{LIM}_{n\to\infty} \frac{m_n - 1}{n}\)。 现在证明 \(S\) 是最小上界。
  1. \(S\) 是上界： 对任意 \(x \in E\)，由于 \(m_n/n\) 是上界， 有 \(x \leq m_n/n\) 对所有 \(n\) 成立。由习题 5.4.8，\(x \leq S\)。
  2. \(S\) 是最小的： 设 \(y\) 是任意上界。由于 \((m_n - 1)/n\) 不是上界， 有 \(y \geq (m_n - 1)/n\) 对所有 \(n\) 成立。由习题 5.4.8，\(y \geq S\)。
因此 \(S\) 即为所求的最小上界。
📖 补充说明：定理 5.5.9 的证明策略

该证明是全书最复杂的证明之一。其核心思想是：

  • 用有理数逼近上确界： 对每个精度 \(1/n\)，找到"最佳"的有理数估计 \(m_n/n\)； 该有理数是上界，但减去 \(1/n\) 后就不再是上界；
  • 证明这些有理数构成 Cauchy 序列： 利用上界/非上界的夹逼得到差值估计；
  • 取形式极限： \(S = \operatorname{LIM} m_n/n\) 就是上确界。

在 Section_5_5.lean 中，该证明被拆分为多个引理： Real.LUB_claim1（存在唯一的 \(m\)）、Real.LUB_claim2（Cauchy 性估计）、 Real.LIM_of_Cauchy（辅助引理），最后在 Real.LUB_exist 中组装。

5.5.10：

5.5.10（）。 设 \(E \subseteq \mathbb{R}\)。
  • 如果 \(E\) 非空且有上界，定义 \(\sup(E)\) 为 \(E\) 的最小上界（由定理 5.5.9 保证存在唯一）；
  • 如果 \(E\) 非空且无上界，定义 \(\sup(E) := +\infty\)；
  • 如果 \(E\) 为空，定义 \(\sup(E) := -\infty\)。
\(\sup(E)\) 称为 \(E\) 的上确界（supremum），也记作 \(\sup E\)。
注 5.5.11。 目前 \(+\infty\) 和 \(-\infty\) 只是无意义的符号；它们不是实数， 不能参与实数运算。在第 6.2 节中我们将把它们加入实数系统构成扩展实数系 （extended real number system），但许多代数定律在此系统中会失效。

5.5.12：\(\sqrt{2}\) 的存在性

5.5.12。 存在正实数 \(x\) 使得 \(x^2 = 2\)。
令 \(E := \{y \in \mathbb{R} : y \geq 0 \text{ 且 } y^2 < 2\}\)。 \(E\) 非空（\(1 \in E\)）且有上界（例如 2 是上界，因为若 \(y > 2\) 则 \(y^2 > 4 > 2\)）。 由上确界性质，设 \(x := \sup(E)\)。则 \(x \geq 1\) 且 \(x \leq 2\)，故 \(x > 0\)。 我们用反证法证明 \(x^2 = 2\)。

情况 1： 假设 \(x^2 < 2\)。取 \(0 < \varepsilon < 1\)，则 \[ (x + \varepsilon)^2 = x^2 + 2\varepsilon x + \varepsilon^2 \leq x^2 + 4\varepsilon + \varepsilon = x^2 + 5\varepsilon. \] 由于 \(x^2 < 2\)，可选择足够小的 \(\varepsilon\) 使得 \(x^2 + 5\varepsilon < 2\)， 从而 \((x + \varepsilon)^2 < 2\)，故 \(x + \varepsilon \in E\)，与 \(x\) 是上界矛盾。

情况 2： 假设 \(x^2 > 2\)。取 \(0 < \varepsilon < 1\)，则 \[ (x - \varepsilon)^2 = x^2 - 2\varepsilon x + \varepsilon^2 \geq x^2 - 4\varepsilon. \] 选择足够小的 \(\varepsilon\) 使得 \(x^2 - 4\varepsilon > 2\)，则 \((x - \varepsilon)^2 > 2\)， 从而 \(x - \varepsilon \geq y\) 对所有 \(y \in E\) 成立（否则若 \(x - \varepsilon < y\)， 则 \((x - \varepsilon)^2 < y^2 \leq 2\) 矛盾）。故 \(x - \varepsilon\) 是 \(E\) 的上界， 与 \(x\) 是最小上界矛盾。

因此 \(x^2 = 2\)。
📖 补充说明：\(\sqrt{2}\) 的存在性与上确界

命题 5.5.12 与命题 4.4.4 形成鲜明对比：有理数中没有 \(\sqrt{2}\)， 但实数中有。正是上确界性质保证了实数中"应该有"的数都存在。 证明中使用的"反证法 + \(\varepsilon\) 调整"是分析学中最基本的技巧之一。

在 Lean 中，该证明需要巧妙地选择 \(\varepsilon\) 并处理多个不等式的放缩。 Real.exist_sqrt_two 的证明使用了 E 作为辅助集合， 通过上确界构造出 \(\sqrt{2}\)，然后对 \(x^2 < 2\) 和 \(x^2 > 2\) 两种情况分别导出矛盾。

📖 补充说明：注 5.5.13–5.5.15

命题 5.5.12 也说明有理数 \(\mathbb{Q}\) 不具有上确界性质—— 否则我们就能用同样的方法在 \(\mathbb{Q}\) 中构造出 \(\sqrt{2}\)， 而命题 4.4.4 告诉我们这是不可能的。

下界和最大下界（下确界 \(\inf\)）的概念完全对称。 \(\inf(E)\) 可以通过 \(\sup(-E)\) 来定义（见习题 5.5.1）。

— 第 5.5 节总结 —

上确界性质是实数区别于有理数的核心特征。有以下几点值得注意：

  1. 它保证了"填补间隙"：\(\sqrt{2}\)、\(\pi\)、\(e\) 等所有无理数的存在性都可由此推出；
  2. 它是后续分析学所有重要定理的基础（介值定理、极值定理、Bolzano-Weierstrass 定理等）；
  3. 它的证明依赖于阿基米德性质（推论 5.4.13）和 Cauchy 序列的构造技巧。
📖 补充说明：上确界与 Mathlib 的 ConditionallyCompleteLattice

习题 5.5.1。 设 \(E \subseteq \mathbb{R}\) 有最小上界 \(M = \sup(E)\)。 令 \(-E := \{-x : x \in E\}\)。证明 \(-M\) 是 \(-E\) 的最大下界，即 \(-M = \inf(-E)\)。

对应 Lean 目标：Real.inf_neg。

习题 5.5.2。 设 \(E \subseteq \mathbb{R}\) 非空，\(n \geq 1\)，\(L < K\) 为整数。 假设 \(K/n\) 是 \(E\) 的上界而 \(L/n\) 不是。不使用定理 5.5.9， 证明存在整数 \(L < m \leq K\) 使得 \(m/n\) 是上界但 \((m-1)/n\) 不是。 （提示：用反证法和归纳法。画图可能有助于理解。）

对应 Lean 目标：Real.upperBound_between。

习题 5.5.3。 设 \(E \subseteq \mathbb{R}\) 非空，\(n \geq 1\)，\(m, m'\) 为整数。 如果 \(m/n\) 和 \(m'/n\) 都是 \(E\) 的上界，但 \((m-1)/n\) 和 \((m'-1)/n\) 都不是， 证明 \(m = m'\)。（因此习题 5.5.2 中的 \(m\) 是唯一的。）

对应 Lean 目标：Real.upperBound_discrete_unique。

习题 5.5.4。 设 \(q_1, q_2, q_3, \dots\) 是一个序列， 满足对任意整数 \(M \geq 1\) 及任意 \(n, n' \geq M\) 有 \(|q_n - q_{n'}| \leq 1/M\)。 证明该序列是 。 此外，若 \(S := \operatorname{LIM} q_n\)，证明对每个 \(M \geq 1\) 有 \(|q_M - S| \leq 1/M\)。 （提示：使用推论 5.4.10 或习题 5.4.8。）

对应 Lean 目标：Real.LIM_of_Cauchy。

习题 5.5.5。 建立命题 5.4.14 的类比版本，将其中"有理数"替换为"无理数"。 即证明：对任意两个实数 \(x < y\)，存在无理数 \(z\) 使得 \(x < z < y\)。

对应 Lean 目标：Real.irrat_between。
