4 整数与有理数

第4.2节 有理数 第69–73页

我们现在已经构造了整数，以及加法、减法、乘法运算和序关系，并验证了所有预期的代数和序理论性质。 现在我们将使用类似的构造来建造有理数，将除法加入我们的运算组合中。

正如整数是通过两个自然数相减构造出来的，有理数可以通过两个整数相除来构造， 不过当然我们必须有通常的警告：分母不能为零。2 当然，正如两个差 \(a-b\) 和 \(c-d\) 可以在 \(a+d=c+b\) 时相等一样， 我们从更高级的知识中知道，两个商 \(a/b\) 和 \(c/d\) 可以在 \(ad=bc\) 时相等。 因此，类比整数，我们创建一个新的无意义符号 //（最终将被除法取代），并定义：

定义 4.2.1（有理数）。 一个有理数是形如 \(a//b\) 的表达式，其中 \(a\) 和 \(b\) 是整数且 \(b\) 非零；\(a//0\) 不被视为有理数。 当且仅当 \(ad = cb\) 时，我们认为两个有理数相等，记作 \(a//b = c//d\)。 所有有理数的集合记作 \(\mathbb{Q}\)。

因此例如 \(3//4 = 6//8 = -3//-4\)，但 \(3//4 \neq 4//3\)。 这是一个合法的相等定义（习题 4.2.1）。 现在我们需要加法、乘法和否定的概念。同样，我们将利用我们已有的知识， 它告诉我们 \(a/b + c/d\) 应等于 \((ad+bc)/(bd)\)， 且 \(a/b \times c/d\) 应等于 \(ac/bd\)，而 \(-(a/b)\) 等于 \((-a)/b\)。 受此先验知识的启发，我们定义：

定义 4.2.2（有理数的加法、乘法与否定）。 如果 \(a//b\) 和 \(c//d\) 是有理数，我们定义它们的和
\[(a//b) + (c//d) := (ad + bc)//(bd),\]
它们的积
\[(a//b) \times (c//d) := (ac)//(bd),\]
以及否定
\[-(a//b) := (-a)//b.\]

注意如果 \(b\) 和 \(d\) 非零，由命题 4.1.8 可知 \(bd\) 也非零，因此两个有理数的和或积仍然是有理数。

引理 4.2.3（和、积与否定运算是良好定义的）。 有理数上的加法、乘法和否定运算是良好定义的，即如果将 \(a//b\) 替换为与之相等的另一个有理数 \(a'//b'\)， 上述运算的输出保持不变；对于 \(c//d\) 也同样成立。
我们只验证加法；其余断言留给习题 4.2.2。 假设 \(a//b = a'//b'\)，因此 \(b\) 和 \(b'\) 非零且 \(ab' = a'b\)。 我们要证明 \(a//b + c//d = a'//b' + c//d\)。 由定义，左边是 \((ad+bc)//bd\)，右边是 \((a'd+b'c)//b'd\)， 因此我们需要证明 \[(ad+bc)b'd = (a'd+b'c)bd,\] 展开得到 \[ab'd^2 + bb'cd = a'bd^2 + bb'cd.\] 但由 \(ab' = a'b\)，结论成立。同样地，如果将 \(c//d\) 替换为 \(c'//d'\) 也成立。

我们注意到，形如 \(a//1\) 的有理数的行为与整数 \(a\) 完全相同：

\[(a//1) + (b//1) = (a+b)//1;\qquad (a//1) \times (b//1) = (ab)//1;\qquad -(a//1) = (-a)//1.\]

此外，\(a//1\) 与 \(b//1\) 相等当且仅当 \(a\) 与 \(b\) 相等。 因此，我们将每个整数 \(a\) 等同于 \(a//1\)：\(a \equiv a//1\)； 上述恒等式保证了整数的算术与有理数的算术是一致的。 正如我们将自然数嵌入整数一样，我们将整数嵌入有理数。 特别地，所有自然数都是有理数，例如 \(0\) 等于 \(0//1\)，\(1\) 等于 \(1//1\)。

注意一个有理数 \(a//b\) 等于 \(0 = 0//1\) 当且仅当 \(a \times 1 = b \times 0\)， 即分子 \(a\) 等于 \(0\)。因此如果 \(a\) 和 \(b\) 都非零，则 \(a//b\) 也非零。

我们现在在有理数上定义一个新的运算：倒数。 如果 \(x = a//b\) 是一个非零有理数（因此 \(a, b \neq 0\)）， 我们定义 \(x\) 的倒数 \(x^{-1}\) 为有理数 \(x^{-1} := b//a\)。 容易验证这个运算与我们的相等概念是一致的： 如果两个有理数 \(a//b\) 与 \(a'//b'\) 相等，那么它们的倒数也相等。 （相比之下，像"分子"这样的运算不是良好定义的：有理数 \(3//4\) 和 \(6//8\) 相等，但分子不相等， 因此我们在引用像"\(x\) 的分子"这样的术语时必须小心。）然而，我们将 \(0\) 的倒数保留为未定义。

现在我们总结有理数的代数性质。

命题 4.2.4（有理数的代数定律）。 设 \(x, y, z\) 是有理数。那么以下代数定律成立：
\[ \begin{aligned} x + y &= y + x \\ (x + y) + z &= x + (y + z) \\ x + 0 = 0 + x &= x \\ x + (-x) = (-x) + x &= 0 \\ xy &= yx \\ (xy)z &= x(yz) \\ x \cdot 1 = 1 \cdot x &= x \\ x(y + z) &= xy + xz \\ (y + z)x &= yx + zx. \end{aligned} \]
如果 \(x\) 非零，我们还有
\[xx^{-1} = x^{-1}x = 1.\]
注 4.2.5。 上述十条恒等式有一个名称；它们断言有理数 \(\mathbb{Q}\) 构成了一个域。 这比成为交换环更好，因为有了第十条恒等式 \(xx^{-1}=x^{-1}x=1\)。 注意本命题取代了命题 4.1.6。
为了证明这些恒等式，将 \(x = a//b\)，\(y = c//d\)，\(z = e//f\)， 其中 \(a, c, e\) 是整数，\(b, d, f\) 是非零整数，然后依次使用整数的代数验证每个恒等式。 我们只证明最长的那个，即 \((x+y)+z = x+(y+z)\)：
\[ \begin{aligned} (x + y) + z &= ((a//b) + (c//d)) + (e//f) \\ &= ((ad+bc)//bd) + (e//f) \\ &= (adf + bcf + bde)//bdf; \\ x + (y + z) &= (a//b) + ((c//d) + (e//f)) \\ &= (a//b) + ((cf+de)//df) \\ &= (adf + bcf + bde)//bdf \end{aligned} \]
由此可见 \((x+y)+z\) 与 \(x+(y+z)\) 相等。其他恒等式的证明类似，留给习题 4.2.3。

现在我们可以定义两个有理数 \(x\) 和 \(y\) 的商 \(x/y\)，前提是 \(y\) 非零：

\[x/y := x \times y^{-1}.\]

例如， \[(3//4)/(5//6) = (3//4) \times (6//5) = (18//20) = (9//10).\] 利用这个公式容易看出，对每个整数 \(a\) 和每个非零整数 \(b\)，有 \(a/b = a//b\)。 因此我们现在可以丢弃 // 记号，使用更常规的 \(a/b\) 代替 \(a//b\)。

类似地，我们定义有理数上的减法：

\[x - y := x + (-y),\]

和我们在整数时所做的完全一样。

命题 4.2.4 允许我们使用所有通常的代数规则；从现在起我们将不加说明地使用它们。

在上一节中，我们将整数分为正数、零和负数。现在我们对有理数做同样的事。

定义 4.2.6（正有理数与负有理性）。 一个有理数 \(x\) 被称为是正的，当且仅当存在正整数 \(a\) 和 \(b\) 使得 \(x = a/b\)。 它被称为是负的，当且仅当存在某个正有理数 \(y\) 使得 \(x = -y\) （即存在正整数 \(a\) 和 \(b\) 使得 \(x = (-a)/b\)）。

因此，例如每个正整数都是正有理数，每个负整数都是负有理性数， 所以我们的新定义与旧定义是一致的。

引理 4.2.7（有理数的三歧性）。 设 \(x\) 是一个有理数。那么以下三条陈述中恰好有一条为真： (a) \(x\) 等于 \(0\)；(b) \(x\) 是正有理数；(c) \(x\) 是负有理性数。
见习题 4.2.4。
定义 4.2.8（有理数的序）。 设 \(x\) 和 \(y\) 是有理数。我们说 \(x > y\) 当且仅当 \(x - y\) 是正有理数， 且 \(x < y\) 当且仅当 \(x - y\) 是负有理性数。 我们记 \(x \geq y\) 当 \(x > y\) 或 \(x = y\)，类似地定义 \(x \leq y\)。
命题 4.2.9（有理数序的基本性质）。 设 \(x, y, z\) 是有理数。那么以下性质成立：
  1. （序的三歧性）\(x = y\)、\(x < y\) 和 \(x > y\) 中恰好有一个为真。
  2. （序是反对称的）\(x < y\) 当且仅当 \(y > x\)。
  3. （序是传递的）如果 \(x < y\) 且 \(y < z\)，则 \(x < z\)。
  4. （加法保持序）如果 \(x < y\)，则 \(x + z < y + z\)。
  5. （正数乘法保持序）如果 \(x < y\) 且 \(z\) 为正，则 \(xz < yz\)。
见习题 4.2.5。
注 4.2.10。 命题 4.2.9 中的上述五条性质，结合命题 4.2.4 中的域公理， 有一个名称：它们断言有理数 \(\mathbb{Q}\) 构成了一个有序域。 重要的是要记住命题 4.2.9(e) 仅在 \(z\) 为正时成立，见习题 4.2.6。
