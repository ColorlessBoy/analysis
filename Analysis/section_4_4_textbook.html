— 引言 —

设想我们将有理数排列在一条直线上，若 \(x > y\) 则将 \(x\) 放在 \(y\) 的右侧。 （这种排列方式并不严格，因为我们尚未定义"直线"的概念， 但以下讨论仅用于引出后面的严格命题。）在有理数内部有整数， 因此整数也以同样的方式排列在这条直线上。现在我们来考察有理数相对于整数的排列方式。

在此前的构造中，我们从皮亚诺公理出发得到自然数 \(\mathbb{N}\)， 通过形式差得到整数 \(\mathbb{Z}\)，再通过形式商得到有理数 \(\mathbb{Q}\)。 每一步都验证了新系统继承了之前的代数运算和序关系。 有理数 \(\mathbb{Q}\) 已在第 4.2–4.3 节中被证实为一个有序域， 其中定义了加法、减法、乘法、除法、序关系、绝对值和整数指数幂运算。

然而，虽然有理数在序下"稠密"（任意两个有理数之间还有有理数）， 有理数系在分析意义上却是"不完备"的——存在"间隙"或"漏洞"。 本节将揭示这些间隙，特别是证明 \(\sqrt{2}\) 不是有理数， 同时说明我们仍然可以用有理数任意逼近 \(\sqrt{2}\)。 这将为下一章构造实数 \(\mathbb{R}\) 提供核心动机。

命题 4.4.1：整数的穿插

命题 4.4.1（有理数对整数的穿插）。 设 \(x\) 是一个有理数。则存在一个整数 \(n\) 使得 \(n \leq x < n + 1\)。事实上，这个整数是唯一的 （即对于每个 \(x\)，恰好存在一个 \(n\) 满足 \(n \leq x < n + 1\)）。 特别地，存在一个自然数 \(N\) 使得 \(N > x\) （即不存在大于所有自然数的有理数）。
注。 命题的前半部分有时被称为阿基米德性质（Archimedean property）的雏形： 对于任何有理数 \(x\)，都存在一个自然数大于它。 这个性质在实数构造中将扮演关键角色。
📖 补充说明：命题 4.4.1 意味着什么？

直觉上，命题 4.4.1 告诉我们以下事实：

  • 整数覆盖了数轴。 任何有理数都落在某个长度为 1 的整数区间 \([n, n+1)\) 中。
  • 整数的无界性。 不存在一个有理数能"超过"所有自然数；自然数本身是无界的。
  • 任意有理数 x 都存在整数部分。 这个唯一的 \(n\) 就是 \(x\) 的"向下取整" \(\lfloor x \rfloor\)。
见习题 4.4.1。（提示：使用命题 2.3.9。）
注 4.4.2（整数部分）。 满足 \(n \leq x < n + 1\) 的整数 \(n\) 有时被称为 \(x\) 的整数部分（integer part）， 并记作 \(n = \lfloor x \rfloor\)。

命题 4.4.3：有理数的稠密性

命题 4.4.3（有理数对有理数的穿插）。 若 \(x\) 和 \(y\) 是两个有理数且满足 \(x < y\)， 则存在第三个有理数 \(z\) 使得 \(x < z < y\)。
📖 补充说明：这个性质为什么叫"稠密性"？

如果一个集合 \(S\) 中任意两个不同元素之间都存在 \(S\) 中的另一个元素， 我们就说 \(S\) 是稠密的（在自身中稠密，或者说"序稠密"）。 命题 4.4.3 正说明了 \(\mathbb{Q}\) 在自身中稠密。对比来看， \(\mathbb{Z}\) 就不是稠密的——在 3 和 4 之间没有其他整数。

但要注意，"稠密"不等于"完备"。有理数虽然有稠密性， 却仍然缺少 \(\sqrt{2}\) 这样的点——这就是所谓"间隙"的由来。

令 \(z := (x + y)/2\)。由于 \(x < y\)，且 \(1/2\) 是正的， 从命题 4.2.9 可得 \(x/2 < y/2\)。向两边同时加 \(y/2\) 得到 \(x/2 + y/2 < y/2 + y/2\)，即 \(z < y\)。向两边同时加 \(x/2\) 得到 \(x/2 + x/2 < y/2 + x/2\)，即 \(x < z\)。因此 \(x < z < y\) 如欲证。
📖 补充说明：命题 4.4.3 在 Lean 中的证明

在 Section_4_4.lean 中，该命题已作为 Rat.exists_between_rat 完成。其证明使用了 \(z := (x+y)/2\) 的构造，以及域的算术和序运算。在 Lean 中， 也可以利用 linarith 或 nlinarith 来简化。

— 稠密但不完备 —

尽管有理数具有上述稠密性质，它们仍然是不完备的； 有理数之间存在无穷多个"间隙"或"漏洞"（gaps / holes）。 不过，稠密性确保这些间隙在某种意义上是"无穷小的"。 例如，我们现在将证明有理数中没有任何一个数的平方等于 2。

命题 4.4.4：\(\sqrt{2}\) 不是有理数

命题 4.4.4。 不存在有理数 \(x\) 使得 \(x^2 = 2\)。
此处仅给出证明的梗概；其中的细节将在习题 4.4.3 中补充。

证明梗概。 假设为了推出矛盾，存在一个有理数 \(x\) 使得 \(x^2 = 2\)。 显然 \(x \neq 0\)。我们可以假定 \(x\) 是正的（若 \(x\) 为负，用 \(-x\) 替代，因为 \(x^2 = (-x)^2\)）。于是 \(x = p/q\)，其中 \(p, q\) 是正整数。因此 \((p/q)^2 = 2\)，整理得 \(p^2 = 2q^2\)。

称自然数 \(p\) 为偶的（even），若 \(p = 2k\) 对某自然数 \(k\) 成立； 称其为奇的（odd），若 \(p = 2k + 1\) 对某自然数 \(k\) 成立。 每个自然数要么是偶的要么是奇的，但不可能同时是两者（为什么？）。 若 \(p\) 是奇的，则 \(p^2\) 也是奇的（为什么？）， 这与 \(p^2 = 2q^2\) 矛盾（因为右侧是偶数）。因此 \(p\) 是偶的， 即 \(p = 2k\) 对某自然数 \(k\) 成立。 由于 \(p\) 为正，\(k\) 也必为正。将 \(p = 2k\) 代入 \(p^2 = 2q^2\) 得 \(4k^2 = 2q^2\)，即 \(q^2 = 2k^2\)。

总结：我们从一组正整数对 \((p, q)\) 满足 \(p^2 = 2q^2\) 出发， 得到了一组新的正整数对 \((q, k)\) 满足 \(q^2 = 2k^2\)。 由于 \(p^2 = 2q^2\)，我们有 \(q < p\)（为什么？）。 记 \(p' := q\)，\(q' := k\)，则我们从一个解 \((p, q)\) 出发， 得到了方程 \(p^2 = 2q^2\) 的一个更小的解 \((p', q')\)（即 \(p' < p\)）。

重复这一过程，我们得到解序列 \((p, q), (p', q'), (p'', q''), \dots\)，每个解中 \(p\) 的值都比前一个小， 且所有分量都是正整数。但这与无穷递降原理 （principle of infinite descent，见习题 4.4.2）矛盾。 这一矛盾表明不可能存在满足 \(x^2 = 2\) 的有理数 \(x\)。

📖 补充说明：无穷递降法

无穷递降法（infinite descent）是 Fermat 发明的一种证明技巧。 其核心想法是：从一组正整数解出发，构造出一组严格更小的正整数解， 重复下去得到无穷递减的正整数序列——这在自然数中是不可能的 （因为自然数有最小元 1，不能无限下降）。这个方法常用于证明某些丢番图方程无解。

在命题 4.4.4 的证明中，我们构造了递减的正整数序列： \[ p > p' > p'' > p''' > \dots \geq 1 \] 这是不可能的。因此原假设（存在有理数 \(x\) 使得 \(x^2 = 2\)）为假。

在 Lean 的 Section_4_4.lean 中，这一证明已实现为 Rat.not_exist_sqrt_two（定理 4.4.4）。该证明巧妙利用了 自然数的奇偶性和无穷递降原理。

📖 补充说明：奇偶性的关键细节

证明中需要用到的几个事实：

  • 偶数的平方是偶数： 若 \(p = 2k\)，则 \(p^2 = 4k^2 = 2(2k^2)\)。
  • 奇数的平方是奇数： 若 \(p = 2k+1\)，则 \(p^2 = 4k^2 + 4k + 1 = 2(2k^2 + 2k) + 1\)。
  • 奇偶排中律： 每个自然数要么是偶数要么是奇数，但不能同时是两者。
  • 从 \(p^2 = 2q^2\) 推出 \(q < p\)： 因为 \(p^2 = 2q^2 > q^2\)（对 \(p, q > 0\)），而正数的平方保持序。

这些细节构成了证明的骨架，也是习题 4.4.3 要求读者补充的"为什么？"。

命题 4.4.5：任意接近 \(\sqrt{2}\) 的有理数

命题 4.4.5。 对每个有理数 \(\varepsilon > 0\)，存在一个非负有理解 \(x\) 使得 \(x^2 < 2 < (x + \varepsilon)^2\)。
设 \(\varepsilon > 0\) 是有理数。假设为了推出矛盾，不存在非负有理解 \(x\) 使得 \(x^2 < 2 < (x + \varepsilon)^2\)。这意味着每当 \(x\) 非负且 \(x^2 < 2\) 时， 必有 \((x + \varepsilon)^2 < 2\)（注意 \((x + \varepsilon)^2\) 不可能等于 2， 由命题 4.4.4）。由于 \(0^2 < 2\)，我们有 \(\varepsilon^2 < 2\)， 进而 \((2\varepsilon)^2 < 2\)，通过简单的归纳可知对每个自然数 \(n\) 有 \((n\varepsilon)^2 < 2\)。（注意对每个自然数 \(n\)，\(n\varepsilon\) 是非负的——为什么？） 但由命题 4.4.1，我们可以找到一个整数 \(n\) 使得 \(n > 2/\varepsilon\)， 即 \(n\varepsilon > 2\)，于是 \((n\varepsilon)^2 > 4 > 2\)，与 "对所有 \(n\) 有 \((n\varepsilon)^2 < 2\)" 矛盾。这一矛盾即完成了证明。
📖 补充说明：命题 4.4.5 的证明策略

该证明使用了一种常见的分析学论证模式：反证法 + 阿基米德性质。 关键在于，命题 4.4.1 保证了可以取到足够大的 \(n\) 使得 \(n\varepsilon > 2\)， 从而构造出矛盾。本质上，该命题说明有理数虽然不包含 \(\sqrt{2}\)， 但在有理数中可以找到任意接近 \(\sqrt{2}\) 的点——\(\sqrt{2}\) 是 有理数集的极限点（limit point）。

在 Lean 中，该命题已实现为 Rat.exist_approx_sqrt_two。

— 走向实数：极限的思想 —

命题 4.4.5 表明，虽然有理数集 \(\mathbb{Q}\) 中确实没有 \(\sqrt{2}\) 这个元素， 但我们可以任意接近 \(\sqrt{2}\)。例如，序列

\[ 1.4,\; 1.41,\; 1.414,\; 1.4142,\; 1.41421,\; \dots \]

看起来越来越接近 \(\sqrt{2}\)，它们的平方也表明这一点：

\[ 1.96,\; 1.9881,\; 1.99396,\; 1.99996164,\; 1.9999899241,\; \dots \]

因此，我们可以通过对有理数序列取"极限"来"创造" \(\sqrt{2}\)。 这正是下一章构造实数的方法。

（另一种方法是使用所谓的"戴德金分割"（Dedekind cuts）， 但本书不采用此路径。还可以使用无穷十进制展开，但这会带来一些棘手的问题， 例如必须处理 \(0.999\dots = 1.000\dots\) 的情况。 虽然这种方法最为人所熟知，但实际上比其他方法更复杂；见附录 B。）

📖 补充说明：第 4.4 节在整个分析学中的地位

第 4.4 节是第 4 章（整数与有理数）的最后一节， 也是通向第 5 章（实数）的桥梁。它的核心信息可以用一句话概括：

"有理数稠密但不完备，实数的构造正是为了填补这些间隙。"

具体来说：

  • 稠密性（命题 4.4.3）：任意两个有理数之间还有有理数。这使 \(\mathbb{Q}\) 看起来"很满"。
  • 不完备性（命题 4.4.4）：虽然稠密，但 \(\mathbb{Q}\) 中缺少 \(\sqrt{2}\) 这样的数。
  • 逼近可能性（命题 4.4.5）：虽然 \(\sqrt{2} \notin \mathbb{Q}\)，但我们可以用有理数任意逼近它。

这三个事实合起来指向了第 5 章的基本思路：将实数定义为有理数 Cauchy 序列的极限。

📖 补充说明：为什么 \(\sqrt{2}\) 的无理性如此重要？

\(\sqrt{2}\) 的无理性（即命题 4.4.4）是数学史上最重要的发现之一。 传说古希腊毕达哥拉斯学派的 Hippasus 证明了这一事实， 而该学派认为"一切皆数"（即一切皆有理数）， 据说 Hippasus 因此被抛入海中。

从现代数学的视角看，\(\sqrt{2}\) 的无理性说明：

  • 有理数系在几何上不够用（边长为 1 的正方形的对角线长为 \(\sqrt{2}\)）。
  • 有理数系不"完备"——在分析意义上，我们需要实数。
  • "无穷递降法"是一个强大的证明技术，可用于许多数论问题。
📖 补充说明：第 4 章整体结构回顾

第 4 章是全书中最具"构造性"风格的一章。我们从自然数 \(\mathbb{N}\)（第 2 章）出发， 逐步构造了：

  • 整数 \(\mathbb{Z}\)（第 4.1 节）： 自然数的形式差 \(a - b\)。
  • 有理数 \(\mathbb{Q}\)（第 4.2 节）： 整数的形式商 \(a // b\)（\(b \neq 0\)）。
  • 绝对值与指数（第 4.3 节）： 在 \(\mathbb{Q}\) 上定义绝对值和整数指数幂。
  • 有理数的间隙（第 4.4 节）： 揭示 \(\mathbb{Q}\) 的局限，motivate 第 5 章。

这种"从简单到复杂"的构造路径，既展示了数学基础的严谨性， 也为读者提供了理解数系扩张的自然视角。

习题 4.4.2. 定义：一个序列 \(a_0, a_1, a_2, \dots\) 称为无穷递降 （infinite descent），如果对每个自然数 \(n\) 都有 \(a_n > a_{n+1}\) （即 \(a_0 > a_1 > a_2 > \dots\)）。

(a) 证明无穷递降原理：不可能存在一个自然数序列处于无穷递降中。 （提示：假设为了推出矛盾，存在一个处于无穷递降的自然数序列。 由于所有 \(a_n\) 都是自然数，故 \(a_n \geq 0\) 对所有 \(n\) 成立。 然后用归纳法证明实际上 \(a_n \geq k\) 对所有 \(k \in \mathbb{N}\) 和所有 \(n \in \mathbb{N}\) 成立， 从而导出矛盾。）

(b) 如果允许序列 \(a_1, a_2, a_3, \dots\) 取整数值而不是自然数值， 无穷递降原理仍然成立吗？如果允许取正有理数值呢？请解释。
