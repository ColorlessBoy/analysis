— 引言 —

设想我们将排列在一条直线上，若 \(x > y\) 则将 \(x\) 放在 \(y\) 的右侧。 （这种排列方式并不严格，因为我们尚未定义"直线"的概念， 但以下讨论仅用于引出后面的严格命题。）在内部有， 因此也以同样的方式排列在这条直线上。现在我们来考察相对于的排列方式。

在此前的构造中，我们从皮亚诺公理出发得到 \(\mathbb{N}\)， 通过得到 \(\mathbb{Z}\)，再通过得到 \(\mathbb{Q}\)。 每一步都验证了新系统继承了之前的代数运算和序关系。 \(\mathbb{Q}\) 已在第 4.2–4.3 节中被证实为一个， 其中定义了加法、减法、乘法、除法、序关系、和运算。

然而，虽然在序下"稠密"（任意两个之间还有）， 系在分析意义上却是""的——存在""或""。 本节将揭示这些，特别是证明 \(\sqrt{2}\) 不是， 同时说明我们仍然可以用任意逼近 \(\sqrt{2}\)。 这将为下一章构造 \(\mathbb{R}\) 提供核心动机。

命题 4.4.1：的穿插

命题 4.4.1（对的穿插）。 设 \(x\) 是一个。则存在一个 \(n\) 使得 \(n \leq x < n + 1\)。事实上，这个整数是唯一的 （即对于每个 \(x\)，恰好存在一个 \(n\) 满足 \(n \leq x < n + 1\)）。 特别地，存在一个自然数 \(N\) 使得 \(N > x\) （即不存在大于所有的）。
注。 命题的前半部分有时被称为（Archimedean property）的雏形： 对于任何 \(x\)，都存在一个大于它。 这个性质在构造中将扮演关键角色。
📖 补充说明：命题 4.4.1 意味着什么？

直觉上，命题 4.4.1 告诉我们以下事实：

  • 覆盖了数轴。 任何都落在某个长度为 1 的区间 \([n, n+1)\) 中。
  • 的无界性。 不存在一个能"超过"所有；本身是无界的。
  • 任意 x 都存在。 这个唯一的 \(n\) 就是 \(x\) 的"向下取整" \(\lfloor x \rfloor\)。
见习题 4.4.1。（提示：使用命题 2.3.9。）
注 4.4.2（）。 满足 \(n \leq x < n + 1\) 的整数 \(n\) 有时被称为 \(x\) 的（integer part）， 并记作 \(n = \lfloor x \rfloor\)。

命题 4.4.3：的

命题 4.4.3（对的穿插）。 若 \(x\) 和 \(y\) 是两个且满足 \(x < y\)， 则存在第三个有理数 \(z\) 使得 \(x < z < y\)。
📖 补充说明：这个性质为什么叫""？

如果一个集合 \(S\) 中任意两个不同元素之间都存在 \(S\) 中的另一个元素， 我们就说 \(S\) 是（在自身中稠密，或者说"序稠密"）。 命题 4.4.3 正说明了 \(\mathbb{Q}\) 在自身中稠密。对比来看， \(\mathbb{Z}\) 就不是——在 3 和 4 之间没有其他。

但要注意，"稠密"不等于"完备"。虽然有， 却仍然缺少 \(\sqrt{2}\) 这样的点——这就是所谓""的由来。

令 \(z := (x + y)/2\)。由于 \(x < y\)，且 \(1/2\) 是正的， 从命题 4.2.9 可得 \(x/2 < y/2\)。向两边同时加 \(y/2\) 得到 \(x/2 + y/2 < y/2 + y/2\)，即 \(z < y\)。向两边同时加 \(x/2\) 得到 \(x/2 + x/2 < y/2 + x/2\)，即 \(x < z\)。因此 \(x < z < y\) 如欲证。

命题 4.4.4：\(\sqrt{2}\) 不是

命题 4.4.4。 不存在 \(x\) 使得 \(x^2 = 2\)。
此处仅给出证明的梗概；其中的细节将在习题 4.4.3 中补充。

证明梗概。 假设为了推出矛盾，存在一个 \(x\) 使得 \(x^2 = 2\)。 显然 \(x \neq 0\)。我们可以假定 \(x\) 是正的（若 \(x\) 为负，用 \(-x\) 替代，因为 \(x^2 = (-x)^2\)）。于是 \(x = p/q\)，其中 \(p, q\) 是正。因此 \((p/q)^2 = 2\)，整理得 \(p^2 = 2q^2\)。

称 \(p\) 为（even），若 \(p = 2k\) 对某 \(k\) 成立； 称其为（odd），若 \(p = 2k + 1\) 对某 \(k\) 成立。 每个要么是要么是，但不可能同时是两者（为什么？）。 若 \(p\) 是，则 \(p^2\) 也是（为什么？）， 这与 \(p^2 = 2q^2\) 矛盾（因为右侧是）。因此 \(p\) 是， 即 \(p = 2k\) 对某 \(k\) 成立。 由于 \(p\) 为正，\(k\) 也必为正。将 \(p = 2k\) 代入 \(p^2 = 2q^2\) 得 \(4k^2 = 2q^2\)，即 \(q^2 = 2k^2\)。

总结：我们从一组正对 \((p, q)\) 满足 \(p^2 = 2q^2\) 出发， 得到了一组新的正对 \((q, k)\) 满足 \(q^2 = 2k^2\)。 由于 \(p^2 = 2q^2\)，我们有 \(q < p\)（为什么？）。 记 \(p' := q\)，\(q' := k\)，则我们从一个解 \((p, q)\) 出发， 得到了方程 \(p^2 = 2q^2\) 的一个更小的解 \((p', q')\)（即 \(p' < p\)）。

重复这一过程，我们得到解 \((p, q), (p', q'), (p'', q''), \dots\)，每个解中 \(p\) 的值都比前一个小， 且所有分量都是正。但这与 （principle of infinite descent，见习题 4.4.2）矛盾。 这一矛盾表明不可能存在满足 \(x^2 = 2\) 的 \(x\)。

📖 补充说明：

（infinite descent）是 Fermat 发明的一种证明技巧。 其核心想法是：从一组正解出发，构造出一组严格更小的正解， 重复下去得到递减的正——这在中是不可能的 （因为有最小元 1，不能无限下降）。这个方法常用于证明某些丢番图方程无解。

在命题 4.4.4 的证明中，我们构造了递减的正： \[ p > p' > p'' > p''' > \dots \geq 1 \] 这是不可能的。因此原假设（存在 \(x\) 使得 \(x^2 = 2\)）为假。

在 Lean 的 Section_4_4.lean 中，这一证明已实现为 Rat.not_exist_sqrt_two（定理 4.4.4）。该证明巧妙利用了 的奇偶性和。

📖 补充说明：奇偶性的关键细节

证明中需要用到的几个事实：

  • 的平方是： 若 \(p = 2k\)，则 \(p^2 = 4k^2 = 2(2k^2)\)。
  • 的平方是： 若 \(p = 2k+1\)，则 \(p^2 = 4k^2 + 4k + 1 = 2(2k^2 + 2k) + 1\)。
  • 奇偶排中律： 每个要么是要么是，但不能同时是两者。
  • 从 \(p^2 = 2q^2\) 推出 \(q < p\)： 因为 \(p^2 = 2q^2 > q^2\)（对 \(p, q > 0\)），而正数的平方保持序。

这些细节构成了证明的骨架，也是习题 4.4.3 要求读者补充的"为什么？"。

命题 4.4.5：任意接近 \(\sqrt{2}\) 的

命题 4.4.5。 对每个 \(\varepsilon > 0\)，存在一个非负有理解 \(x\) 使得 \(x^2 < 2 < (x + \varepsilon)^2\)。
设 \(\varepsilon > 0\) 是。假设为了推出矛盾，不存在非负有理解 \(x\) 使得 \(x^2 < 2 < (x + \varepsilon)^2\)。这意味着每当 \(x\) 非负且 \(x^2 < 2\) 时， 必有 \((x + \varepsilon)^2 < 2\)（注意 \((x + \varepsilon)^2\) 不可能等于 2， 由命题 4.4.4）。由于 \(0^2 < 2\)，我们有 \(\varepsilon^2 < 2\)， 进而 \((2\varepsilon)^2 < 2\)，通过简单的归纳可知对每个自然数 \(n\) 有 \((n\varepsilon)^2 < 2\)。（注意对每个自然数 \(n\)，\(n\varepsilon\) 是非负的——为什么？） 但由命题 4.4.1，我们可以找到一个整数 \(n\) 使得 \(n > 2/\varepsilon\)， 即 \(n\varepsilon > 2\)，于是 \((n\varepsilon)^2 > 4 > 2\)，与 "对所有 \(n\) 有 \((n\varepsilon)^2 < 2\)" 矛盾。这一矛盾即完成了证明。
📖 补充说明：命题 4.4.5 的证明策略

该证明使用了一种常见的分析学论证模式： + 。 关键在于，命题 4.4.1 保证了可以取到足够大的 \(n\) 使得 \(n\varepsilon > 2\)， 从而构造出矛盾。本质上，该命题说明虽然不包含 \(\sqrt{2}\)， 但在中可以找到任意接近 \(\sqrt{2}\) 的点——\(\sqrt{2}\) 是 集的（limit point）。

在 Lean 中，该命题已实现为 Rat.exist_approx_sqrt_two。

— 走向：的思想 —

命题 4.4.5 表明，虽然集 \(\mathbb{Q}\) 中确实没有 \(\sqrt{2}\) 这个元素， 但我们可以任意接近 \(\sqrt{2}\)。例如，

\[ 1.4,\; 1.41,\; 1.414,\; 1.4142,\; 1.41421,\; \dots \]

看起来越来越接近 \(\sqrt{2}\)，它们的平方也表明这一点：

\[ 1.96,\; 1.9881,\; 1.99396,\; 1.99996164,\; 1.9999899241,\; \dots \]

因此，我们可以通过对取""来"创造" \(\sqrt{2}\)。 这正是下一章构造的方法。

（另一种方法是使用所谓的""（Dedekind cuts）， 但本书不采用此路径。还可以使用十进制展开，但这会带来一些棘手的问题， 例如必须处理 \(0.999\dots = 1.000\dots\) 的情况。 虽然这种方法最为人所熟知，但实际上比其他方法更复杂；见附录 B。）

📖 补充说明：第 4.4 节在整个分析学中的地位

第 4.4 节是第 4 章（与）的最后一节， 也是通向第 5 章（）的桥梁。它的核心信息可以用一句话概括：

"稠密但，的构造正是为了填补这些。"

具体来说：

  • （命题 4.4.3）：任意两个之间还有。这使 \(\mathbb{Q}\) 看起来"很满"。
  • 性（命题 4.4.4）：虽然稠密，但 \(\mathbb{Q}\) 中缺少 \(\sqrt{2}\) 这样的数。
  • 逼近可能性（命题 4.4.5）：虽然 \(\sqrt{2} \notin \mathbb{Q}\)，但我们可以用任意逼近它。

这三个事实合起来指向了第 5 章的基本思路：将定义为 Cauchy 的。

📖 补充说明：为什么 \(\sqrt{2}\) 的无理性如此重要？

\(\sqrt{2}\) 的无理性（即命题 4.4.4）是数学史上最重要的发现之一。 传说古希腊毕达哥拉斯学派的 Hippasus 证明了这一事实， 而该学派认为"一切皆数"（即一切皆）， 据说 Hippasus 因此被抛入海中。

从现代数学的视角看，\(\sqrt{2}\) 的无理性说明：

  • 系在几何上不够用（边长为 1 的正方形的对角线长为 \(\sqrt{2}\)）。
  • 系不"完备"——在分析意义上，我们需要。
  • ""是一个强大的证明技术，可用于许多数论问题。
📖 补充说明：第 4 章整体结构回顾

第 4 章是全书中最具"构造性"风格的一章。我们从 \(\mathbb{N}\)（第 2 章）出发， 逐步构造了：

  • \(\mathbb{Z}\)（第 4.1 节）： 的 \(a - b\)。
  • \(\mathbb{Q}\)（第 4.2 节）： 的 \(a // b\)（\(b \neq 0\)）。
  • 与指数（第 4.3 节）： 在 \(\mathbb{Q}\) 上定义和。
  • 的（第 4.4 节）： 揭示 \(\mathbb{Q}\) 的局限，motivate 第 5 章。

这种"从简单到复杂"的构造路径，既展示了数学基础的严谨性， 也为读者提供了理解数系扩张的自然视角。
