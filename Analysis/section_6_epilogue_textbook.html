— 引言 —

在这个技术性的尾声中，我们证明本章为"第 6 章"序列 Chapter6.Sequence 所定义的各种运算和性质， 与 Mathlib 中的对应运算是等价的。需要注意的是，Mathlib 的运算在远比实值序列更一般的框架下定义， 特别地使用了的语言。

本节是一个技术性章节 (technical chapter)。它不再引入新的数学概念， 而是把"教材的 \(\varepsilon\)-\(N\) 框架"与"Mathlib 的滤框架"之间的等价关系严格化。 原书中并无对应章节——这是 Lean 形式化项目独有的"桥接层"。

从第 6 章开始，本项目已从第 5 章的 Chapter5.Real 切换到 Mathlib 的 ℝ（参见第 5 章尾声的同构 equivR）。 但本章中序列的概念（Chapter6.Sequence）、收敛（TendsTo）、Cauchy 性（IsCauchy）、 有界性（IsBounded）等仍是项目自定义的，与 Mathlib 的 CauSeq、Filter.Tendsto、 Bornology.IsBounded 等并行存在。本尾声的目的就是消除这种"双轨并行"—— 证明二者等价，从而后续章节可以直接调用 Mathlib 的丰富 API。

— Cauchy 序列的等价 —

教材中 Cauchy 序列的定义使用 \(\varepsilon\)-\(N\) 语言：对每个 \(\varepsilon > 0\)， 存在 \(N\) 使得对所有 \(n, m \geq N\) 有 \(|a_n - a_m| \leq \varepsilon\)。 Mathlib 中 IsCauSeq 和 CauchySeq 用类似但稍有不同的方式表达同一概念。

补充说明：Lean 中的桥接定理
/-- Cauchy ↔ Mathlib's IsCauSeq (abs version) -/ theorem Chapter6.Sequence.isCauchy_iff_isCauSeq (a: ℕ → ℝ) : (a:Sequence).IsCauchy ↔ IsCauSeq _root_.abs a /-- Cauchy ↔ Mathlib's CauchySeq -/ theorem Chapter6.Sequence.Cauchy_iff_CauchySeq (a: ℕ → ℝ) : (a:Sequence).IsCauchy ↔ CauchySeq a

证明的核心是将两边的 \(\varepsilon\)-\(N\) 量化展开后逐项对应。 IsCauSeq 用 \(\varepsilon/2\) 的技巧对接，CauchySeq 则通过 isCauSeq_iff_cauchySeq 转化。一个技术细节：CauSeq.IsComplete ℝ 在 Mathlib 中是对 abs 建立的，本尾声补充了对 norm 的实例：

instance inst_real_complete : CauSeq.IsComplete ℝ norm := by convert Real.instIsCompleteAbs
— 收敛性的等价（滤语言） —

教材中收敛的定义是：\((a_n)\) 收敛到 \(L\)，如果对每个 \(\varepsilon > 0\)，存在 \(N\) 使得对所有 \(n \geq N\) 有 \(|a_n - L| \leq \varepsilon\)。 Mathlib 用滤语言表达：atTop.Tendsto a (nhds L)，即序列 \(a\) 沿 atTop 滤收敛到 \(L\) 的邻域。

补充说明：Lean 中的桥接定理
/-- TendsTo ↔ Mathlib's Filter.Tendsto (function version) -/ theorem Chapter6.Sequence.tendsto_iff_Tendsto (a: ℕ → ℝ) (L:ℝ) : (a:Sequence).TendsTo L ↔ atTop.Tendsto a (nhds L) /-- TendsTo ↔ Mathlib's Filter.Tendsto (Sequence type version) -/ theorem Chapter6.Sequence.tendsto_iff_Tendsto' (a: Sequence) (L:ℝ) : a.TendsTo L ↔ atTop.Tendsto a.seq (nhds L) /-- Convergent ↔ ∃ L, Filter.Tendsto -/ theorem Chapter6.Sequence.converges_iff_Tendsto (a: ℕ → ℝ) : (a:Sequence).Convergent ↔ ∃ L, atTop.Tendsto a (nhds L)

证明通过 Metric.tendsto_atTop 将滤语言翻译回 \(\varepsilon\)-\(N\) 语言， 然后与教材定义逐项对应。整数与自然数下标之间的转换（Int.toNat 等）是主要的技术细节。

— 极限的等价 —

教材定义收敛序列的极限为 Chapter6.lim，而 Mathlib 有 CauSeq.lim 和 limUnder。 本尾声证明它们给出相同的值。

补充说明：Lean 中的桥接定理
/-- Chapter6.lim = CauSeq.lim -/ theorem Chapter6.Sequence.lim_eq_CauSeq_lim (a:ℕ → ℝ) (ha: (a:Sequence).IsCauchy) : Chapter6.lim (a:Sequence) = CauSeq.lim ⟨ a, (isCauchy_iff_isCauSeq a).mp ha⟩ /-- Chapter6.lim = limUnder atTop -/ theorem Chapter6.Sequence.lim_eq_limUnder (a:ℕ → ℝ) (ha: (a:Sequence).Convergent) : Chapter6.lim (a:Sequence) = limUnder Filter.atTop a

lim_eq_CauSeq_lim 利用 CauSeq.tendsto_limit 和 lim_def 将两边都翻译成 Tendsto，再用收敛极限的唯一性（tendsTo_unique）。 lim_eq_limUnder 类似，利用 tendsto_nhds_limUnder 和滤极限唯一性。

— 有界性的等价 —
补充说明：Lean 中的桥接定理
/-- IsBounded ↔ Bornology.IsBounded (range) -/ theorem Chapter6.Sequence.isBounded_iff_isBounded_range (a:ℕ → ℝ): (a:Sequence).IsBounded ↔ Bornology.IsBounded (Set.range a)

教材的 IsBounded 定义为存在 \(M \geq 0\) 使所有 \(|a_n| \leq M\) （通过 boundedBy）。Mathlib 的 Bornology.IsBounded 定义为值域的直径有界。证明利用 \(|a_n - a_m| \leq |a_n| + |a_m| \leq 2M\)（正向）， 以及 \(|a_n| \leq |a_n - a_0| + |a_0| \leq C + |a_0|\)（反向）。

— 上确界/下确界的等价 —
补充说明：Lean 中的桥接定理
/-- sup = sSup (range as EReal) -/ theorem Chapter6.Sequence.sup_eq_sSup (a:ℕ → ℝ): (a:Sequence).sup = sSup (Set.range (fun n ↦ (a n:EReal))) /-- inf = sInf (range as EReal) -/ theorem Chapter6.Sequence.inf_eq_sInf (a:ℕ → ℝ): (a:Sequence).inf = sInf (Set.range (fun n ↦ (a n:EReal))) /-- BddAbove ↔ Mathlib's BddAbove -/ theorem Chapter6.Sequence.bddAbove_iff (a:ℕ → ℝ): (a:Sequence).BddAbove ↔ _root_.BddAbove (Set.range a) /-- BddBelow ↔ Mathlib's BddBelow -/ theorem Chapter6.Sequence.bddBelow_iff (a:ℕ → ℝ): (a:Sequence).BddBelow ↔ _root_.BddBelow (Set.range a)

教材的 Sequence.sup 定义为拓展实数 EReal 中集合 \(\{a_n : n \geq m\}\) 的上确界。桥接定理将其与 Mathlib 的 Set.range 的 sSup 等同。关键步骤是证明两个集合相等 （整数下标 vs 自然数下标，利用 Int.toNat）。

— 单调性的等价 —
补充说明：Lean 中的桥接定理
/-- IsMonotone ↔ Mathlib's Monotone -/ theorem Chapter6.Sequence.Monotone_iff (a:ℕ → ℝ): (a:Sequence).IsMonotone ↔ Monotone a /-- IsAntitone ↔ Mathlib's Antitone -/ theorem Chapter6.Sequence.Antitone_iff (a:ℕ → ℝ): (a:Sequence).IsAntitone ↔ Antitone a

教材的 IsMonotone 定义为"对所有 \(n \geq m\) 有 \(a_{n+1} \geq a_n\)"（逐项递增）， 而 Mathlib 的 Monotone 定义为"对所有 \(n \leq m\) 有 \(a_n \leq a_m\)"（全局单调）。 二者等价：逐项递增通过归纳法推出全局单调，反之取 \(m = n+1\) 即得逐项。 IsAntitone 与 Antitone 类似。

— 极限点的等价 —

教材中极限点 \(L\) 的定义是：对每个 \(\varepsilon > 0\) 和每个 \(N\)，存在 \(n \geq N\) 使得 \(|a_n - L| \leq \varepsilon\)。 Mathlib 中对应的概念是 MapClusterPt L atTop a——序列 \(a\) 沿 atTop 滤在 \(L\) 处聚簇。

补充说明：Lean 中的桥接定理
/-- LimitPoint ↔ MapClusterPt -/ theorem Chapter6.Sequence.limit_point_iff (a: ℕ → ℝ) (L:ℝ) : (a:Sequence).LimitPoint L ↔ MapClusterPt L .atTop a

证明将 MapClusterPt 展开为 frequently（"无穷经常"）， 再展开为 ∃ᶠ n in atTop, P n（"对任意 \(N\)，存在 \(n \geq N\) 使 \(P n\) 成立"）， 这恰好就是教材中极限点的定义。邻域用 Metric.ball L ε 翻译。

滤语言的优势在于它统一了"序列收敛"、"函数在一点连续"、"函数在无穷远处的行为"等看似不同的概念， 用同一个 Tendsto 谓词表达。教材的 \(\varepsilon\)-\(N\) 框架则更直观、更适合教学。 桥接定理让两种语言可以自由切换。

个人感悟：桥接层的价值与代价

本尾声的存在反映了一个形式化项目中的核心张力：忠实教材 vs 利用标准库。 教材用 \(\varepsilon\)-\(N\) 语言定义一切，因为它面向人类读者，追求直观与可读性。 Mathlib 用滤和拓扑语言定义一切，因为它面向机器和一般性，追求统一与复用。

桥接定理的价值是双向的：一方面，它们让教材中证明的定理（如 Bolzano–Weierstrass、单调收敛等） 可以"免费"翻译成 Mathlib 的语言，被后续章节调用；另一方面，Mathlib 的丰富 API （如 tendsto_nhds_unique、continuousAt、各种自动化的 simp 引理集） 也可以反过来辅助教材定理的证明——第 6.7 节中 Real.rpow_eq_rpow 的证明就是一个典型例子： 它利用了 Mathlib 的连续性定理来避免重新证明一遍"取极限"的技术细节。

代价是：桥接定理本身需要写，而且往往涉及繁琐的类型转换（ℕ → ℝ vs Sequence， 整数下标 vs 自然数下标，EReal vs ℝ）。但这是一次性的投入—— 一旦桥接完成，"两个世界"就合为一体。从第 7 章起，序列层面的工作可以直接调用 Mathlib 的拓扑 API， 而不必每次都回到 \(\varepsilon\)-\(N\) 的第一性原理。

— 第 6 章尾声·完 —

本章证明了第 6 章中为序列定义的各种概念——Cauchy 性、收敛、有界性、上确界/下确界、单调性、极限点—— 均与 Mathlib 的标准概念等价。结合第 6.7 节中实指数运算与 Mathlib 指数的同一性， 第 6 章的工作已完全融入 Mathlib 的分析框架。从第 7 章起，我们将直接使用 Mathlib 的 API。
