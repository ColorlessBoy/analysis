4 整数与有理数

第4.1节 整数 第63–68页

在第二章中，我们建立了自然数系的大部分基本性质，但我们已经达到了仅靠加法和乘法所能做的极限。 现在我们想引入一个新的运算——减法，但要正确地做到这一点，我们必须从自然数系过渡到一个更大的数系，即整数系。

直观地说，整数是我们可以从两个自然数相减得到的数；例如，3 − 5 应该是整数，6 − 2 也应该是整数。 但这不是一个完整的定义，因为 (a) 它没有说明何时两个差相等（例如，我们应该知道为什么 3 − 5 等于 2 − 4，但又不等于 1 − 6），以及 (b) 它没有说明如何对这些差进行算术运算（例如，如何将 3 − 5 加上 6 − 2？）。 此外，(c) 这个定义是循环的，因为它需要减法的概念，而减法只有在整数构造完成后才能充分定义。

幸运的是，由于我们之前的整数经验，我们知道这些问题的答案应该是什么。 对于 (a)，我们从代数知识中知道，a − b = c − d 当且仅当 a + d = c + b，因此我们可以用加法来刻画差的相等性。 类似地，对于 (b)，我们从代数中知道

\[(a - b) + (c - d) = (a + c) - (b + d)\]

and that

\[(a - b)(c - d) = (ac + bd) - (ad + bc).\]

所以我们将利用我们的预知，将所有这些内容纳入整数的定义中，我们稍后就会这样做。

我们仍然要解决 (c)。为了绕过这个问题，我们使用以下变通方法：我们暂时不将整数写成差 a − b，而是使用新的符号 a — b 来定义整数，其中 — 是一个无意义的占位符，类似于笛卡尔坐标表示 (x, y) 中使用的逗号。 稍后当我们定义减法时，我们将看到 a — b 实际上等于 a − b，因此我们可以丢弃 — 这个符号；它现在只是需要用来避免循环性。 （这些技巧类似于建造建筑物时使用的脚手架；它们暂时是必不可少的，以确保建筑物正确建造，但一旦建筑物建成，它们就会被拆除，不再使用。）

定义 4.1.1（整数）。 整数是形如 a — b 的表达式1，其中 a 和 b 是自然数。 当且仅当
\[a + d = c + b,\]
时，我们认为两个整数相等，记作 a — b = c — d。 我们用 \(\mathbb{Z}\) 表示所有整数的集合。

因此，例如 3—5 是一个整数，并且等于 2—4，因为 3+4 = 2+5。 另一方面，3—5 不等于 2—3，因为 3+3 ≠ 2+5。 这种符号看起来很奇怪，而且有一些缺陷；例如，3 还不是整数，因为它不是 a — b 的形式！我们稍后将纠正这些问题。

我们必须检查这是否是一个合法的相等概念。我们需要验证自反性、对称性、传递性和替换公理（见 A.7 节）。 我们把自反性和对称性留给习题 4.1.1，这里只验证传递性公理。 假设我们知道 a—b = c—d 和 c—d = e—f。那么我们有 a+d = c+b 和 c+f = d+e。 将这两个等式相加得到

\[a + d + c + f = c + b + d + e.\]

根据命题 2.2.6，我们可以消去 c 和 d，得到 a + f = b + e，即 a—b = e—f。 因此需要消去律来确保我们的相等概念是合理的。

定义 4.1.2（加法与乘法）。 两个整数的和 \((a-b) + (c-d)\) 定义为
\[(a-b) + (c-d) := (a+c) - (b+d).\]
两个整数的积 \((a - b) \times (c-d)\) 定义为
\[(a-b) \times (c-d) := (ac+bd) - (ad+bc).\]
1 用集合论的语言来说，我们在这里所做的是从自然数有序对 (a,b) 的空间 \(\mathbb{N} \times \mathbb{N}\) 出发。 然后我们在这些有序对上放置一个等价关系 ∼，规定当且仅当 a + d = c + b 时，(a,b) ∼ (c,d)。 符号 a — b 的集合论解释是：它是与 (a,b) 等价的所有有序对的空间，即 a — b := {(c,d) ∈ N×N : (a,b) ∼ (c,d)}； 整数的集合 \(\mathbb{Z} = \{a-b : (a,b) \in \mathbb{N}\times\mathbb{N}\}\) 的存在性随后可以从两次应用替换公理得出。
引理 4.1.3（加法和乘法是良好定义的）。 设 a, b, a', b', c, d 是自然数。如果 \((a - b) = (a' - b')\)，那么 \((a - b) + (c-d) = (a'-b') + (c-d)\) 和 \((a-b) \times (c-d) = (a'-b') \times (c-d)\)， 同样也有 \((c-d) + (a-b) = (c-d) + (a'-b')\) 和 \((c-d) \times (a-b) = (c-d) \times (a'-b')\)。 因此加法和乘法是良好定义的运算（相同的输入给出相同的输出）。
为了证明 \((a-b)+(c-d) = (a'-b')+(c-d)\)，我们分别计算两侧为 \((a+c)-(b+d)\) 和 \((a'+c)-(b'+d)\)。 因此我们需要证明 \(a+c+b'+d = a'+c+b+d\)。 但由于 \((a-b) = (a'-b')\)，我们有 \(a+b' = a'+b\)，于是两边加上 \(c+d\) 即得所证。

整数 \(n - 0\) 的行为与自然数 n 相同；事实上可以验证 \((n-0)+(m-0) = (n+m)-0\) 和 \((n-0) \times (m-0) = nm - 0\)。 此外，\((n-0)\) 等于 \((m-0)\) 当且仅当 \(n = m\)。（数学术语称此为自然数 n 与形如 \(n - 0\) 的整数之间存在同构。） 因此我们可以通过设 \(n \equiv n - 0\) 来将自然数与整数等同起来；这不会影响我们的加法、乘法或相等的定义，因为它们彼此是一致的。 例如，自然数 3 现在被认为是与整数 3—0 相同的，因此 \(3 = 3 - 0\)。特别地，0 等于 \(0-0\)，1 等于 \(1-0\)。

定义 4.1.4（整数的否定）。 如果 \((a-b)\) 是一个整数，我们定义它的否定 \(-(a-b)\) 为整数 \((b-a)\)。 特别是，如果 \(n = n-0\) 是一个正自然数，我们可以定义它的否定 \(-n = 0-n\)。

例如，\(-(3-5) = (5-3)\)。可以验证这个定义是良好定义的（见习题 4.1.2）。

引理 4.1.5（整数的三歧性）。 设 x 是一个整数。那么以下三条陈述中恰好有一条为真：
  1. x 为零；
  2. x 等于某个正自然数 n；
  3. x 是某个正自然数 n 的否定 −n。
我们首先证明 (a)、(b)、(c) 中至少有一个为真。 由定义，x = a—b，其中 a, b 是自然数。 我们有三种情况：a > b, a = b, 或 a < b。 若 a > b，则 a = b + c，其中 c 是某个正自然数，这意味着 a—b = c—0 = c，即为 (b)。 若 a = b，则 a—b = a—a = 0—0 = 0，即为 (a)。 若 a < b，则 b > a，因此 b—a = n（由前面的推理可得某个自然数 n），于是 a—b = −n，即为 (c)。

如果 n 是正自然数，我们称 n 为正整数，而 −n 为负整数。 因此每个整数要么是正的，要么是零，要么是负的，但不能同时属于多于一种情况。

命题 4.1.6（整数的代数定律）。 设 x, y, z 是整数。那么我们有
\[ \begin{aligned} x + y &= y + x \\ (x + y) + z &= x + (y + z) \\ x + 0 = 0 + x &= x \\ x + (-x) = (-x) + x &= 0 \\ xy &= yx \\ (xy)z &= x(yz) \\ x \cdot 1 = 1 \cdot x &= x \\ x(y + z) &= xy + xz \\ (y + z)x &= yx + zx. \end{aligned} \]
注 4.1.7。 上述九个恒等式组有一个名称；它们声称整数构成了一个交换环。 （如果删去 xy = yx，那么它们只声称整数构成了一个环。）

现在我们定义两个整数 \(x - y\) 的减法运算为

\[x - y := x + (-y).\]

我们不需要为这个运算验证替换公理，因为我们已经将减法用整数的另外两个运算——加法和否定——来定义了，而我们已经验证过那些运算是良好定义的。

命题 4.1.8（整数没有零因子）。 设 a 和 b 是整数，使得 ab = 0。那么要么 a = 0，要么 b = 0（或两者同时为零）。
推论 4.1.9（整数的消去律）。 如果 a, b, c 是整数且 ac = bc，且 c 不为零，则 a = b。
定义 4.1.10（整数的序）。 设 n 和 m 是整数。我们说 n 大于或等于 m，记作 \(n \geq m\) 或 \(m \leq n\)，当且仅当存在某个自然数 a 使得 \(n = m + a\)。 我们说 n 严格大于 m，记作 \(n > m\) 或 \(m < n\)，当且仅当 \(n \geq m\) 且 \(n \neq m\)。
引理 4.1.11（序的性质）。 设 a, b, c 是整数。
  1. a > b 当且仅当 a − b 是正自然数。
  2. （加法保持序）若 a > b，则 a + c > b + c。
  3. （正数乘法保持序）若 a > b 且 c 为正，则 ac > bc。
  4. （取负反转序）若 a > b，则 −a < −b。
  5. （序的传递性）若 a > b 且 b > c，则 a > c。
  6. （序的三歧性）a > b, a < b, 或 a = b 中恰好有一个为真。
