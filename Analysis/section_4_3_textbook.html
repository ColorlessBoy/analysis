— 引言 —

在前三节中，我们从皮亚诺公理出发构造了自然数 \(\mathbb{N}\)， 然后通过形式差构造了整数 \(\mathbb{Z}\)，又通过形式商构造了有理数 \(\mathbb{Q}\)。 每一个阶段，我们都验证了新系统继承了上一阶段的所有代数运算（加法、乘法） 和序关系，并获得了新的运算能力。有理数 \(\mathbb{Q}\) 的特殊之处在于它同时拥有 加法、减法、乘法和除法（除数非零），这使它成为一个有序域。

有了有理数的代数结构和序结构之后，本节将引入两个紧密关联的 分析学基本工具：绝对值和指数运算。 它们将在后续章节中用于定义极限、连续性、可微性等核心概念。

📖 补充说明：我们走到了哪里？—— 从群到有序域

在继续学习第 4.3 节之前，有必要回顾一下我们迄今为止构造的代数结构。 这个问题非常重要，因为 Tao 在书中多次提到"群""环""域""有序域"这些术语， 初读时容易产生混淆。下面我们用通俗的语言逐一解释。

1. 半群 (Semigroup) — 只有结合律的加法

自然数 \(\mathbb{N}\) 上的加法满足结合律 \((a+b)+c = a+(b+c)\)， 这就是一个加法半群。半群只需要一个二元运算满足结合律， 不要求有零元、不要求有逆元。

2. 幺半群 (Monoid) — 有零元的半群

\(\mathbb{N}\) 加上 0 以后，有了加法的单位元（零元 0）：\(a+0 = 0+a = a\)。 这就是一个加法幺半群。但自然数仍然没有"负数"， 所以 3 - 5 在 \(\mathbb{N}\) 中无意义。

3. 群 (Group) — 每个元素都有逆元

定义：一个群是一个集合 \(G\) 配上一个二元运算 \(\cdot\)（常写为乘法或加法），满足：

  • 结合律：\((a \cdot b) \cdot c = a \cdot (b \cdot c)\)；
  • 单位元存在：存在 \(e \in G\) 使得 \(e \cdot a = a \cdot e = a\)；
  • 逆元存在：对每个 \(a \in G\)，存在 \(a^{-1} \in G\) 使得 \(a \cdot a^{-1} = a^{-1} \cdot a = e\)。

如果用加法记号，逆元叫"相反数"（写作 \(-a\)），群叫加法群。 整数 \(\mathbb{Z}\) 就是一个加法群：每个整数 \(n\) 都有相反数 \(-n\) 满足 \(n+(-n)=0\)。 自然数 \(\mathbb{N}\) 不是群（缺少逆元），但整数 \(\mathbb{Z}\) 是。

如果运算还满足交换律 \(a \cdot b = b \cdot a\)，就称为交换群 （或 Abel 群）。\(\mathbb{Z}\) 在加法下是交换群。

4. 环 (Ring) — 有两个运算（加法和乘法）

一个环是一个集合 \(R\) 配有两种运算 \(+\) 和 \(\times\)，满足：

  • \(R\) 在加法下是交换群（有零元 0，每个元素有相反数）；
  • 乘法满足结合律和分配律：\(a\times(b+c)=a\times b + a\times c\)；
  • 乘法有单位元 1（幺环）。

整数 \(\mathbb{Z}\) 构成一个交换环（乘法也交换），但它不是域 —— 因为 \(2\) 没有乘法逆元（\(1/2\) 不是整数）。 Tao 在命题 4.1.6 中验证了整数构成环，在注 4.1.7 中指出它是交换环。

5. 域 (Field) — 除法总可行的环

一个域是一个交换环 \(F\)，额外要求：每个非零元素 \(a \neq 0\) 都有乘法逆元 \(a^{-1}\)，满足 \(a \times a^{-1} = 1\)。

有理数 \(\mathbb{Q}\) 就是一个域：任意非零有理数 \(x = a/b\) 有倒数 \(x^{-1} = b/a\)。这就是 Tao 在命题 4.2.4 中验证的十条恒等式， 并在注 4.2.5 中正式宣布 \(\mathbb{Q}\) 是一个域。

结构运算比上一个多了什么？例子
半群1个，满足结合律—\(\mathbb{N},+\)
幺半群同上有单位元\(\mathbb{N}\cup\{0\},+\)
群同上每个元素有逆元\(\mathbb{Z},+\)
环2个（+ 和 ×）加法群 + 乘法 + 分配律\(\mathbb{Z}\)
域2个（+ 和 ×）非零元有乘法逆元\(\mathbb{Q}\)

6. 有序域 (Ordered Field) — 有序关系的域

一个有序域是一个域 \(F\)，其元素可以比较大小（有全序关系 \(\leq\)）， 并且序与域运算相容：

  • 加法保持序：若 \(a \leq b\)，则 \(a+c \leq b+c\)；
  • 正数乘法保持序：若 \(a \leq b\) 且 \(c \geq 0\)，则 \(a\times c \leq b\times c\)。

有理数 \(\mathbb{Q}\) 就是一个有序域——Tao 在注 4.2.10 中正式宣布了这一结论。 实数 \(\mathbb{R}\) 也是有有序域，但整数 \(\mathbb{Z}\) 不是（因为不是域）， 自然数 \(\mathbb{N}\) 更不是（既不是群也不是域）。

为什么这些概念重要？

本节（4.3节）的主题——绝对值——正是有序域上的概念。 绝对值定义依赖于：

  • 域的否定运算（\(-x\)）：来自加法群结构；
  • 序关系（\(x > 0,\; x < 0\)）：来自有序域结构；
  • 乘法：绝对值满足 \(|xy| = |x||y|\)，这是乘法和序共同作用的结果。

因此，理解第 4.3 节内容的真正背景是：我们正在研究有序域 \(\mathbb{Q}\) 上 绝对值和指数运算的性质。这些性质对任何有序域都成立，但我们目前只在 \(\mathbb{Q}\) 上验证它们。

定义 4.3.1（绝对值）。 如果 \(x\) 是一个有理数，\(x\) 的绝对值 \(|x|\) 定义如下： 如果 \(x\) 是正的，则 \(|x| := x\)；如果 \(x\) 是负的，则 \(|x| := -x\)； 如果 \(x\) 是零，则 \(|x| := 0\)。
注。 绝对值的直观含义是"数轴上到原点的距离"。\(|5| = 5\)， \(|-5| = 5\)，两者到 0 的距离都是 5。绝对值总是非负的。
📖 补充说明：绝对值定义的"三分情况"

定义 4.3.1 用三种情况定义了 \(|x|\)：正、负、零。这利用了有理数的 三歧性（引理 4.2.7）：每个有理数要么是零、要么是正、要么是负， 三者恰好居其一。因此定义对所有 \(x\) 都给出了唯一确定的值。

在 Lean 中，我们可以用 if ... then ... else 来表达这种分情况定义：

def abs (x : &rational;) : &rational; := if x > 0 then x else (if x < 0 then -x else 0)

绝对值的一个直接应用是定义两个数之间的距离：

\[ d(x, y) := |x - y|. \]

例如 \(d(3, 5) = |3-5| = |-2| = 2\)，\(d(5, 3) = |5-3| = |2| = 2\)。 距离是对称的——从 3 到 5 的距离等于从 5 到 3 的距离。

命题 4.3.3：绝对值与距离的基本性质

设 \(x, y, z\) 是有理数。

(a) 绝对值的非退化性。 \(|x| \geq 0\)。此外，\(|x| = 0\) 当且仅当 \(x = 0\)。

📖 补充说明：什么是"非退化性"？

"非退化"（non-degenerate）意味着绝对值不会把非零元素映为零。 更一般地说，一个函数 \(f\) 是"非退化的"当 \(f(x)=0\) 蕴含 \(x=0\)。 绝对值 \(| \cdot |\) 正好满足这个性质：\(|x|=0\) 当且仅当 \(x=0\)。 这保证了距离函数 \(d(x,y) = |x-y|\) 能够区分不同的点 （\(d(x,y)=0 \iff x=y\)）。

(b) 绝对值的三角不等式。 \(|x + y| \leq |x| + |y|\)。

𝓖 补充说明：三角不等式为什么叫"三角"？

在平面几何中，三角形任意两边之和大于第三边。对于数轴上的点， "从 \(0\) 到 \(x+y\) 的距离"不超过"从 \(0\) 到 \(x\) 的距离" 加上"从 \(0\) 到 \(y\) 的距离"。这就是一维的三角不等式。 它是分析学中最重要的不等式之一，后面证明极限、连续性的几乎所有性质 都要用到它。

(c) 绝对值与不等式的关系。 \(-y \leq x \leq y\) 当且仅当 \(y \geq |x|\)。 特别地，我们有 \(-|x| \leq x \leq |x|\)。

𝓖 补充说明：这个等价关系为什么有用？

性质 (c) 在分析中极其常见，因为它将绝对值不等式转化为了 普通的双向不等式。例如：

  • \(|x - 3| \leq 0.01\) 等价于 \(-0.01 \leq x - 3 \leq 0.01\)；
  • 即 \(2.99 \leq x \leq 3.01\)；
  • 这告诉我们 \(x\) 落在一个以 3 为中心、半径为 0.01 的区间内。

这种技巧在后面的极限定义中将反复出现：\(|a_n - L| < \varepsilon\) 等价于 \(L - \varepsilon < a_n < L + \varepsilon\)。

(d) 绝对值的乘法性质。 \(|xy| = |x|\,|y|\)。 特别地，\(|-x| = |x|\)。

𝓖 补充说明：绝对值与乘法

\(|xy| = |x||y|\) 意味着绝对值"可乘出去"。这个性质加上三角不等式 \(|x+y| \leq |x|+|y|\)，表明绝对值是一种乘法赋值 （multiplicative valuation）或称绝对值（absolute value） 在域上的结构。

\(|-x| = |x|\) 是 \(|xy|=|x||y|\) 的特例（取 \(y=-1\)，注意 \(|-1|=1\)）。 它表明绝对值关于正负号是对称的：一个数和它的相反数有相同的绝对值。

(e) 距离的非退化性。 \(d(x, y) \geq 0\) 且 \(d(x, y) = 0\) 当且仅当 \(x = y\)。

(f) 距离的对称性与三角不等式。 \(d(x, y) = d(y, x)\)； \(d(x, z) \leq d(x, y) + d(y, z)\)。

𝓖 补充说明：距离函数的三个公理

性质 (e) 和 (f) 合起来表明 \(d(x, y) := |x - y|\) 满足 度量空间（metric space）的三条公理：

  • 正定性：\(d(x,y) \geq 0\)，且 \(d(x,y)=0 \iff x=y\)；
  • 对称性：\(d(x,y) = d(y,x)\)；
  • 三角不等式：\(d(x,z) \leq d(x,y) + d(y,z)\)。

任何满足这三条性质的函数都可以称为"距离"。因此 \((\mathbb{Q}, d)\) 构成了 一个度量空间——这是第五章构造实数 \(\mathbb{R}\) 的基础框架。 第 5 章中，我们将用 Cauchy 序列来"填充"这个度量空间中的"空隙"。

见习题 4.3.1。 （提示：虽然所有这些断言都可以通过分情况讨论来证明——比如 \(x\) 为正、为负或为零——但命题中有些部分可以不用这样繁复的分类 就证明出来。例如可以利用前面部分来证明后面的部分。）
定义 4.3.4（\(\varepsilon\)-接近）。 设 \(\varepsilon > 0\) 是一个有理数，\(x, y\) 是有理数。 我们说 \(y\) 是 \(\varepsilon\)-接近 于 \(x\)，如果 \(d(y, x) \leq \varepsilon\)。
注 4.3.5。 这个定义在数学教科书中并不是标准定义； 我们将用它作为"脚手架"来构造后面更重要的极限（以及 Cauchy 序列）的概念。一旦我们有了那些更高级的概念， 我们将丢弃 \(\varepsilon\)-接近这个概念。
𝓖 补充说明：\(\varepsilon\) 的由来和为什么用希腊字母

\(\varepsilon\)（epsilon）和 \(\delta\)（delta）是分析学中最著名的两个希腊字母。 传统上用 \(\varepsilon\) 表示"任意小的正数"，用 \(\delta\) 表示相应选取的 "足够小的正数"。这个传统可以追溯到 Cauchy 和 Weierstrass 的 \(\varepsilon\)-\(\delta\) 语言。

Tao 在这里定义的 \(\varepsilon\)-接近，本质上是极限定义的"预演"：

  • \(\varepsilon\)-接近：\(|x - y| \leq \varepsilon\)（两个数之间的距离不超过 \(\varepsilon\)）；
  • 序列收敛：\(\lim a_n = L\) 意味着对于任意 \(\varepsilon > 0\)， 存在 \(N\) 使得 \(n \geq N\) 时 \(|a_n - L| \leq \varepsilon\)；
  • Cauchy 序列：对于任意 \(\varepsilon > 0\)，存在 \(N\) 使得 \(m, n \geq N\) 时 \(|a_m - a_n| \leq \varepsilon\)。

可以看到，\(\varepsilon\)-接近是后面所有收敛概念的"原子单元"。

例 4.3.6。

  • 0.99 和 1.01 是 0.1-接近的（因为 \(|0.99 - 1.01| = 0.02 \leq 0.1\)）；
  • 但它们不是 0.01-接近的（因为 \(0.02 > 0.01\)）；
  • 2 和 2 对每个正 \(\varepsilon\) 都是 \(\varepsilon\)-接近的（因为 \(|2-2| = 0 \leq \varepsilon\)）。

命题 4.3.7：\(\varepsilon\)-接近的性质

设 \(x, y, z, w\) 是有理数。

(a) 若 \(x = y\)，则对每个 \(\varepsilon > 0\)，\(x\) 与 \(y\) 是 \(\varepsilon\)-接近的。 反过来，若 \(x\) 与 \(y\) 对每个 \(\varepsilon > 0\) 都是 \(\varepsilon\)-接近的，则 \(x = y\)。

𝓖 补充说明：性质 (a) 与相等性的关系

性质 (a) 表明 \(\varepsilon\)-接近在 \(\varepsilon \to 0\) 的极限下"趋近于"相等。 粗略地说，如果两个数在任意精度下都无法区分，那么它们就是相等的。 这是分析学里一个反复出现的思想：要证明两个数相等，只需证明它们的距离 可以小于任意正数。

(b) 设 \(\varepsilon > 0\)。若 \(x\) 与 \(y\) 是 \(\varepsilon\)-接近的，则 \(y\) 与 \(x\) 也是。

(c) 设 \(\varepsilon, \delta > 0\)。若 \(x\) 与 \(y\) 是 \(\varepsilon\)-接近的，且 \(y\) 与 \(z\) 是 \(\delta\)-接近的，则 \(x\) 与 \(z\) 是 \((\varepsilon + \delta)\)-接近的。

𝓖 补充说明：性质 (b)(c) 与 "近似相等"

Tao 在注 4.3.8 中指出，性质 (a)(b)(c) 应该与相等的自反性、对称性、传递性 公理做比较。"相等"是精确的关系，"\(\varepsilon\)-接近"是其近似版本：

  • 自反性（每个事物等于自身）对应 (a) 的前半部分；
  • 对称性（若 \(a=b\) 则 \(b=a\)）对应 (b)；
  • 传递性（若 \(a=b\) 且 \(b=c\) 则 \(a=c\)）对应 (c)，但误差累加了： 从 \(\varepsilon + \delta\) 而非简单的 \(\varepsilon\)。

(d) 设 \(\varepsilon, \delta > 0\)。若 \(x, y\) 是 \(\varepsilon\)-接近的，且 \(z, w\) 是 \(\delta\)-接近的，则 \(x+z\) 与 \(y+w\) 是 \((\varepsilon+\delta)\)-接近的， 且 \(x-z\) 与 \(y-w\) 也是 \((\varepsilon+\delta)\)-接近的。

𝓖 补充说明：加法和减法保持"近似"

性质 (d) 说的是：如果你对近似相等的数做加法或减法，结果的误差是原来误差的和。 这是极限运算满足加减法的理论基础。后面我们会看到 \(\lim (a_n + b_n) = \lim a_n + \lim b_n\)——这个性质的根源就在这里。

(e) 设 \(\varepsilon > 0\)。若 \(x\) 与 \(y\) 是 \(\varepsilon\)-接近的， 则对任意 \(\varepsilon' > \varepsilon\)，它们也是 \(\varepsilon'\)-接近的。

(f) 设 \(\varepsilon > 0\)。若 \(y\) 和 \(z\) 都与 \(x\) 是 \(\varepsilon\)-接近的， 且 \(w\) 在 \(y\) 和 \(z\) 之间（即 \(y \leq w \leq z\) 或 \(z \leq w \leq y\)）， 则 \(w\) 也与 \(x\) 是 \(\varepsilon\)-接近的。

(g) 设 \(\varepsilon > 0\)。若 \(x\) 与 \(y\) 是 \(\varepsilon\)-接近的， 且 \(z\) 非零，则 \(xz\) 与 \(yz\) 是 \(\varepsilon|z|\)-接近的。

(h) 设 \(\varepsilon, \delta > 0\)。若 \(x\) 与 \(y\) 是 \(\varepsilon\)-接近的， 且 \(z\) 与 \(w\) 是 \(\delta\)-接近的，则 \(xz\) 与 \(yw\) 是 \((\varepsilon|z| + \delta|x| + \varepsilon\delta)\)-接近的。

我们只证明最困难的一个，(h)；其余留给习题 4.3.2。

设 \(\varepsilon, \delta > 0\)，且 \(x\) 与 \(y\) 是 \(\varepsilon\)-接近的。 令 \(a := y - x\)，则 \(y = x + a\) 且 \(|a| \leq \varepsilon\)。 类似地，若 \(z\) 与 \(w\) 是 \(\delta\)-接近的，令 \(b := w - z\)， 则 \(w = z + b\) 且 \(|b| \leq \delta\)。

由于 \(y = x + a\) 且 \(w = z + b\)，我们有 \[ yw = (x + a)(z + b) = xz + az + xb + ab. \] 因此 \[ |yw - xz| = |az + bx + ab| \leq |az| + |bx| + |ab| = |a||z| + |b||x| + |a||b|. \] 由于 \(|a| \leq \varepsilon\) 且 \(|b| \leq \delta\)，得到 \[ |yw - xz| \leq \varepsilon|z| + \delta|x| + \varepsilon\delta, \] 即 \(xz\) 与 \(yw\) 是 \((\varepsilon|z| + \delta|x| + \varepsilon\delta)\)-接近的。

注 4.3.8。 应将该命题中的陈述 (a)–(c) 与相等关系的自反、对称和传递 公理进行比较。在分析学中，将 \(\varepsilon\)-接近的概念视为"相等"的近似替代 常常是很有用的。
𝓖 补充说明：(h) 的证明为什么那么长？

性质 (h) 是乘法情形下的误差传播。加法时误差简单相加 \((\varepsilon + \delta)\)， 而乘法时误差依赖于乘数本身的大小（出现了 \(|z|\)，\(|x|\)，和交叉项 \(\varepsilon\delta\)）。 这个公式是后面证明 \(\lim (a_n b_n) = (\lim a_n)(\lim b_n)\) 的理论基础。证明的思路——引入辅助变量 \(a = y-x\) 和 \(b = w-z\)——是分析学中的 常见技巧：将要比较的差分解为已知误差的组合。

定义 4.3.9（自然数次幂）。 设 \(x\) 是一个有理数。定义 \(x\) 的 0 次幂为 \(x^0 := 1\)； 特别地，我们定义 \(0^0 := 1\)。现在归纳地假设对某个自然数 \(n\) 已经定义了 \(x^n\)，则定义 \(x^{n+1} := x^n \times x\)。
𝓖 补充说明：指数运算与群结构

指数运算 \(x^n\)（\(n\) 是自然数）本质上是幺半群中的重复乘法。 在任意幺半群 \((M, \cdot, 1)\) 中，我们都可以通过归纳定义 \(a^n\)： \(a^0 = 1\)，\(a^{n+1} = a^n \cdot a\)。这里我们只用到有理数乘法幺半群 \((\mathbb{Q}, \times, 1)\)。

注意 \(0^0 = 1\) 这个约定看起来奇怪，但从组合学和幺半群理论的角度却很自然： 在指数为 0 时，结果是乘法的单位元 1。

命题 4.3.10：指数运算的性质 I

设 \(x, y\) 是有理数，\(n, m\) 是自然数。

(a) \(x^n x^m = x^{n+m}\)，\((x^n)^m = x^{nm}\)，\((xy)^n = x^n y^n\)。

𝓖 补充说明：指数律——为什么同底数的幂可以合并指数？

这三条是指数律（laws of exponents）。用通俗的语言：

  • \(x^n x^m = x^{n+m}\)：相乘时指数相加；
  • \((x^n)^m = x^{nm}\)：幂的幂是指数相乘；
  • \((xy)^n = x^n y^n\)：积的幂等于幂的积。

这些性质源于指数运算的递归定义，可以通过数学归纳法证明。 例如 \(x^n x^m = x^{n+m}\)：先固定 \(n\)，对 \(m\) 归纳；或者固定 \(m\)，对 \(n\) 归纳。

(b) 若 \(n > 0\)，则 \(x^n = 0\) 当且仅当 \(x = 0\)。

𝓖 补充说明：零点与幂

性质 (b) 说的是非零元的幂不为零。在域中，非零元的乘积仍为非零元 （这在命题 4.2.4 的域公理中有所体现），因此重复相乘也不会变成零。 这个性质的另一面是：零的任意正次幂是零。

(c) 若 \(x \geq y \geq 0\)，则 \(x^n \geq y^n \geq 0\)。 若 \(x > y \geq 0\) 且 \(n > 0\)，则 \(x^n > y^n \geq 0\)。

𝓖 补充说明：序与指数运算——为什么正数的幂保持序？

性质 (c) 反映了有序域的一个重要特征：如果 \(x \geq y \geq 0\)，那么 乘以 \(x\) 或 \(y\) 多次不会反转不等式方向。这是由 命题 4.2.9(e)（正数乘法保持序）通过归纳得到的直接推论。

(d) \(|x^n| = |x|^n\)。

见习题 4.3.3。（提示：使用归纳法。）
定义 4.3.11（负整数次幂）。 设 \(x\) 是一个非零有理数。则对任意负整数 \(-n\)， 我们定义 \(x^{-n} := 1/x^n\)。
𝓖 补充说明：负指数扩展了群结构

当指数扩展到全体整数（正、负、零）时，指数运算从幺半群结构升级为了 群结构。具体来说，非零有理数在乘法下构成一个群 \((\mathbb{Q}\setminus\{0\}, \times, 1)\)，而指数运算 \(n \mapsto x^n\) 成为了从加法群 \(\mathbb{Z}\) 到这个乘法群的群同态： \[x^{n+m} = x^n \cdot x^m, \quad x^0 = 1, \quad x^{-n} = (x^n)^{-1}.\]

这就是命题 4.3.12(a) 的群论本质。而我们额外有的就是域上的序结构， 它使得命题 4.3.12(b)(c) 成立。

例如 \(x^{-3} = 1/x^3 = 1/(x \times x \times x)\)。注意当 \(n = 1\) 时，\(x^{-1}\) 的定义与第 4.2 节中定义的倒数一致，所以符号上没有冲突。

我们现在对任何整数 \(n\)（无论正、负或零）都定义了 \(x^n\)。

命题 4.3.12：指数运算的性质 II

设 \(x, y\) 是非零有理数，\(n, m\) 是整数。

(a) \(x^n x^m = x^{n+m}\)，\((x^n)^m = x^{nm}\)，\((xy)^n = x^n y^n\)。

𝓖 补充说明：整数指数与群同态

命题 4.3.12(a) 是将命题 4.3.10(a) 从自然数扩展到全体整数。 证明不是简单的归纳法——需要用命题 4.3.10 的结论来处理涉及负指数的情况。 在群论的语言中，这三条分别表达了指数映射是群同态、 迭代运算的一致性以及直积上的同态。

(b) 若 \(x \geq y > 0\)，则当 \(n\) 为正时 \(x^n \geq y^n > 0\)， 当 \(n\) 为负时 \(0 < x^n \leq y^n\)。 注意负指数会反转不等式方向！

𝓖 补充说明：负指数导致不等式反转

为什么负指数会反转不等式？因为 \(x^{-n} = 1/x^n\)。 如果 \(x > y > 0\)，则 \(x^n > y^n > 0\)（正指数保持序）， 取倒数后序反转：\(1/x^n < 1/y^n\)，即 \(x^{-n} < y^{-n}\)。

例如：\(3 > 2 > 0\)。正指数：\(3^2 = 9 > 2^2 = 4\)。 负指数：\(3^{-2} = 1/9 \approx 0.111 < 2^{-2} = 1/4 = 0.25\)。

(c) 若 \(x, y > 0\)，\(n \neq 0\)，且 \(x^n = y^n\)，则 \(x = y\)。

𝓖 补充说明：正数的指数映射是单射

性质 (c) 说在正有理数范围内，指数函数 \(x \mapsto x^n\)（\(n \neq 0\)） 是单射（injective）——不同的输入给出不同的输出。这个性质在下一章 定义 \(n\) 次方根 \(\sqrt[n]{x}\) 时将非常关键。

(d) \(|x^n| = |x|^n\)。

见习题 4.3.4。（提示：归纳法在这里不适用。应当使用命题 4.3.10。）
𝓖 补充说明：有序域中的绝对值——大格局

现在我们可以把第 4.3 节的所有内容放在一个更大的框架中理解了。

绝对值公理。 在任意有序域 \(F\) 上， 我们可以定义绝对值 \(|x|\) 为： \[ |x| = \begin{cases} x, & x \geq 0, \\ -x, & x < 0. \end{cases} \] 这个定义与 Tao 的定义 4.3.1 完全一致（只是用 \(\geq\) 代替了三分情况）。 它自动满足命题 4.3.3 中的所有性质——这些证明只用到了域公理和序公理， 不需要知道元素具体是什么。

距离与度量空间。 一旦有了绝对值，就可以定义距离 \(d(x,y) = |x-y|\)。这使 \(F\) 成为一个度量空间。

从 \(\mathbb{Q}\) 到 \(\mathbb{R}\)。 Tao 选择先构造 \(\mathbb{Q}\)（一个具体的有序域），然后在上面定义 绝对值和距离，再用 Cauchy 序列构造 \(\mathbb{R}\)（一个完备的有序域）。 这是典型的"从小到大"的构造路径。另一种路径是"抽象公理化"： 直接假设存在一个完备有序域 \(\mathbb{R}\)，然后证明 \(\mathbb{Q}\) 是其子域—— 这两种方式在数学上等价。

第 4 章的层次结构总结。

节构造对象代数结构新运算
4.1整数 \(\mathbb{Z}\)交换环减法
4.2有理数 \(\mathbb{Q}\)有序域除法
4.3绝对值/指数有序域（延续）绝对值、指数
4.4有理数的间隙——（动机章节）

从群到有序域的逐步升级，每一步都增加了一个运算能力（减法、除法、序关系）， 同时保留了上一步的所有性质。这就是"代数结构的层级体系"。 理解这个体系对于后续学习实分析、抽象代数和泛函分析都有重要帮助。
