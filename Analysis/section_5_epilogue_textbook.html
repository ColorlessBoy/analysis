— 引言 —

第 5 章从出发， 用构造出一个 具备结构的对象 \(\mathbf{R}_{\text{Cau}}\)（在 Lean 形式化中记为 Chapter5.Real）。 而 Mathlib 中早已存在一个标准的实数对象 \(\mathbb{R}\)（在 Lean 中记为 ℝ）， 它是用或同等方式构造的。

本章尾声的目标是： 证明这两种实数本质上是同一个对象。 更精确地说，我们要构造一个 \[ \mathrm{equivR} : \mathbf{R}_{\text{Cau}} \;\xrightarrow{\;\sim\;}\; \mathbb{R} \] 作为， 且与有理数嵌入相容、与一切自然/整数/有理指数相容。 有了这个同构后，我们就可以"忘掉"自己费力构造的 \(\mathbf{R}_{\text{Cau}}\)， 直接使用 Mathlib 的 \(\mathbb{R}\) 及其完整的代数与分析 API。

本节是一个技术性章节 (technical chapter)。它不再引入新的数学概念， 而是把"两种构造给出同一个数系"这件直觉上显然的事情严格化。 原书中并无对应章节——这是 Lean 形式化项目独有的"同构桥"。
— 戴德金分割：桥梁 —

为了把 \(\mathbf{R}_{\text{Cau}}\) 和 \(\mathbb{R}\) 这两个看似不相关的对象联系起来， 我们让它们都对应到第三个对象—— \(\mathcal{C}\)。 关键观察是：任何一个有理数 都可以"被它所超越的有理数集合"完全刻画。

（戴德金分割）。 一个 戴德金分割 (Dedekind cut) 是一个有理数子集 \(E \subseteq \mathbb{Q}\)， 满足以下四条：
  1. 非空 (nonempty)：\(E \neq \varnothing\)；
  2. 有上界 (bounded above)：存在 \(M \in \mathbb{Q}\) 使得 \(\forall q \in E, q \leq M\)；
  3. 下集 (lower set / downward closed)：若 \(q \in E\) 且 \(r < q\)，则 \(r \in E\)；
  4. 无最大元 (no maximum)：\(\forall q \in E, \exists r \in E, r > q\)。
用 \(\mathcal{C}\) 表示所有戴德金分割构成的集合。
直观。 把 \(E\) 想象成数轴上"某个实数 \(x\) 之下的全体有理数"， 即 \(E = \{q \in \mathbb{Q} : q < x\}\)。 四条公理逐条对应： 一定有有理数小于 \(x\)（非空）； 一定有有理数大于 \(x\)（有上界）； 比 \(x\) 小的有理数下面也都比 \(x\) 小（下集）； 没有"最大的小于 \(x\) 的有理数"（无最大元，这把 \(x\) 自己排除掉）。
📖 为什么要"无最大元"？

考虑分割 \(E = \{q \in \mathbb{Q} : q \leq 1\}\) 和 \(E' = \{q \in \mathbb{Q} : q < 1\}\)。 如果不要求"无最大元"，这两者会是不同的分割，但它们都"代表" \(1\) 这个实数——一个对应的实数会有两个分割。 要求无最大元就把 \(E\) 排除掉，确保 "实数 \(\leftrightarrow\) 分割"是双射（每个实数恰有一个分割）。

这与 Tao 教科书中 Cauchy 序列定义里要求严格 "\(\varepsilon\)-接近" 而不是 \(\leq\) 的精神类似：略加苛刻一点，换取唯一性。

— 第 5 章实数侧的双射 —

我们先在 \(\mathbf{R}_{\text{Cau}}\) 一侧做戴德金分割的对应。

从 \(\mathbf{R}_{\text{Cau}}\) 出发：取所有"被自己严格超越"的有理数

定义（toCut）。 对每个 \(x \in \mathbf{R}_{\text{Cau}}\)，定义 \[ E_x \;:=\; \{\, q \in \mathbb{Q} \;:\; (q : \mathbf{R}_{\text{Cau}}) < x \,\}. \] 这里 \((q : \mathbf{R}_{\text{Cau}})\) 是把有理数 \(q\) 通过有理数嵌入视为第 5 章实数。
（toCut 是分割）。 对任何 \(x \in \mathbf{R}_{\text{Cau}}\)，集合 \(E_x\) 是一个戴德金分割。
四条公理逐一验证：
  • 非空： 由"两个实数之间存在有理数"（命题 5.4 的结论）， 取 \(q \in (x-1, x)\) 即可，则 \(q < x\)。
  • 有上界： 由， 存在自然数 \(N\) 使 \(x < N\)。则 \(N\) 即 \(E_x\) 的有理数上界。
  • 下集： 若 \(r < q\) 在 \(\mathbb{Q}\) 中，则在 \(\mathbf{R}_{\text{Cau}}\) 中也成立， 从而 \(r < q < x\)，故 \(r \in E_x\)。
  • 无最大元： 对 \(q \in E_x\)（即 \(q < x\)），由有理稠密性， 存在 \(r \in (q, x) \cap \mathbb{Q}\)，故 \(r \in E_x\) 且 \(r > q\)。

从分割出发：取上确界

定义（toReal）。 对每个戴德金分割 \(c\)，定义 \[ \widehat{c} \;:=\; \sup_{\mathbf{R}_{\text{Cau}}} \{\, (q : \mathbf{R}_{\text{Cau}}) \,:\, q \in c \,\}. \] 这里的上确界存在，因为根据命题 5.5（最小上界性质）， \(\mathbf{R}_{\text{Cau}}\) 是的， 并且 \(c\) 在转化为实数集后非空且有上界。
（互逆）。 映射 \(x \mapsto E_x\) 与 \(c \mapsto \widehat{c}\) 互为逆映射。即：
  1. 对任何 \(x \in \mathbf{R}_{\text{Cau}}\)，\(\widehat{E_x} = x\)；
  2. 对任何分割 \(c \in \mathcal{C}\)，\(E_{\widehat{c}} = c\)。
(i) 关键事实： "每个实数都是它所超越的有理数们的"。 证明：
  • 上界： 集合中每一项 \((q : \mathbf{R}_{\text{Cau}})\)（其中 \(q < x\)）显然 \(\leq x\)。
  • 最小性： 假设存在 \(y < x\) 也是上界。由有理稠密性，存在 \(q \in (y, x) \cap \mathbb{Q}\)。 则 \(q < x\) 故 \(q \in E_x\)，但 \(y < q\) 矛盾于 \(y\) 是上界。
(ii) 关键事实： 一个分割可以从它的上确界恢复出来。证明：固定有理数 \(q\)。
  • (\(\Rightarrow\)) 若 \(q \in c\)，由"无最大元"取 \(r \in c\) 且 \(r > q\)。 则 \((q:\mathbf{R}_{\text{Cau}}) < (r:\mathbf{R}_{\text{Cau}}) \leq \widehat{c}\)，故 \(q \in E_{\widehat{c}}\)。
  • (\(\Leftarrow\)) 若 \(q \notin c\)，由"下集"性质，每个 \(r \in c\) 必满足 \(r \leq q\) （否则 \(q < r\) 配合 \(r \in c\) 推出 \(q \in c\)），故 \((q:\mathbf{R}_{\text{Cau}})\) 是 \(c\) 对应的实数集的上界， 从而 \(\widehat{c} \leq q\)，即 \(q \notin E_{\widehat{c}}\)。

综上，我们得到了第 5 章实数侧的双射 \[ \mathrm{equivCut}_{\mathbf{R}_{\text{Cau}}} \;:\; \mathbf{R}_{\text{Cau}} \;\xrightarrow{\;\sim\;}\; \mathcal{C}. \]

— Mathlib 实数侧的双射 —

对 Mathlib 实数 \(\mathbb{R}\) 重复完全相同的构造，逐字替换： \((q : \mathbf{R}_{\text{Cau}})\) 换成 \((q : \mathbb{R})\)， \(\mathbf{R}_{\text{Cau}}\) 中的上确界换成 \(\mathbb{R}\) 中的上确界 (sSup)， \(\mathbf{R}_{\text{Cau}}\) 的有理稠密性换成 Mathlib 的 exists_rat_btwn， 阿基米德性质换成 Mathlib 的 exists_rat_gt。

（\(\mathbb{R}\) 侧的双射）。 映射 \[ x \mapsto E_x^{\mathbb{R}} := \{q \in \mathbb{Q} : (q:\mathbb{R}) < x\},\qquad c \mapsto \widehat{c}^{\,\mathbb{R}} := \sup_{\mathbb{R}}\{(q:\mathbb{R}) : q \in c\} \] 给出双射 \[ \mathrm{equivCut}_{\mathbb{R}} \;:\; \mathbb{R} \;\xrightarrow{\;\sim\;}\; \mathcal{C}. \]
结构上的相似。 两次证明在结构上是一样的： 把"完备性"、"阿基米德性"、"有理稠密性"这三个条件抽出来， \(\mathbf{R}_{\text{Cau}}\) 和 \(\mathbb{R}\) 都满足，自然得到对称的结论。 在更高观点下，这是说"任何完备阿基米德有序域都同构于戴德金分割"， 而我们这里把一般定理实例化了两次。
— 主同构 —

有了两侧分别到 \(\mathcal{C}\) 的双射，把它们组合起来就得到主同构：

（主同构 equivR）。 复合 \[ \mathrm{equivR} \;:=\; \mathrm{equivCut}_{\mathbf{R}_{\text{Cau}}} \;\circ\; \bigl(\mathrm{equivCut}_{\mathbb{R}}\bigr)^{-1} \;:\; \mathbf{R}_{\text{Cau}} \;\xrightarrow{\;\sim\;}\; \mathbb{R} \] 是一个集合双射。其作用是：把第 5 章实数 \(x\) 送到具有"相同分割"的标准实数。
📖 "相同分割"是什么意思

展开看，\(y = \mathrm{equivR}(x)\) 当且仅当 \[ \{q \in \mathbb{Q} : (q:\mathbb{R}) < y\} \;=\; \{q \in \mathbb{Q} : (q:\mathbf{R}_{\text{Cau}}) < x\}. \] 这是一个非常自然的对应：两边都用"哪些有理数严格小于我"来标识自己。

在 Lean 形式化中这一刻画就是引理 Real.equivR_iff : y = equivR x ↔ y.toCut = x.toCut。

— 桥接引理 —

仅仅得到一个集合双射是远远不够的。我们还需要证明它和两侧的代数、序、嵌入都相容。 这部分是本章的技术核心。第一步是处理有理数的嵌入。

有理数嵌入相容

（equivR_ratCast）。 对任何有理数 \(q\)， \[ \mathrm{equivR}\bigl((q : \mathbf{R}_{\text{Cau}})\bigr) \;=\; (q : \mathbb{R}). \]
根据 equivR_iff，只需证 \((q : \mathbb{R})\) 和 \((q : \mathbf{R}_{\text{Cau}})\) 给出相同的分割。 对任意有理数 \(r\)： \[ (r : \mathbb{R}) < (q : \mathbb{R}) \;\Longleftrightarrow\; r < q \;\Longleftrightarrow\; (r : \mathbf{R}_{\text{Cau}}) < (q : \mathbf{R}_{\text{Cau}}). \] 第一和第三个等价是有理数嵌入对序的保持，分别在 \(\mathbb{R}\) 和 \(\mathbf{R}_{\text{Cau}}\) 中成立。

作为推论，自然数和整数的嵌入也相容（因为它们都是有理数嵌入的特例）：

\(\mathrm{equivR}((n:\mathbb{N})) = (n:\mathbb{R}),\qquad \mathrm{equivR}((n:\mathbb{Z})) = (n:\mathbb{R}).\)

两套柯西序列定义的桥接

在 \(\mathbf{R}_{\text{Cau}}\) 中，一个柯西序列 \(a : \mathbb{N} \to \mathbb{Q}\) 通过 \(\mathrm{LIM}\) 给出实数 \(\mathrm{LIM}\, a\)。 而在 Mathlib 中，\(\mathbb{R}\) 是用 CauSeq ℚ abs（对绝对值的有理数序列） 的来构造的，相应的极限算子叫 Real.mk。

我们需要一座翻译之桥：把第 5 章用的 Sequence.IsCauchy 翻译成 Mathlib 用的 IsCauSeq abs， 把第 5 章用的 Sequence.Equiv 翻译成 Mathlib 用的 CauSeq.LimZero。

（柯西性的两种说法）。 若 \(a : \mathbb{N} \to \mathbb{Q}\) 在第 5 章定义意义下是柯西的（\(\forall \varepsilon > 0\) 最终 \(\varepsilon\)-稳定）， 则它在 Mathlib 定义意义下也是关于绝对值的柯西序列。
给定 \(\varepsilon > 0\)。对 \(\varepsilon/2\) 应用柯西性，得 \(N\) 使 \(\forall j, k \geq N, |a_j - a_k| \leq \varepsilon/2\)。 特别地，固定 \(k = N\)，则 \(\forall j \geq N, |a_j - a_N| \leq \varepsilon/2 < \varepsilon\)。
（等价的两种说法）。 若 \(a, b\) 都柯西且 \(a \equiv b\)（第 5 章意义下），则 \(a - b \to 0\)（Mathlib 意义下）。反过来也成立。
第 5 章的等价说"\(\forall \varepsilon > 0\) 最终 \(\varepsilon\)-接近"， Mathlib 的 LimZero 说"\(\forall \varepsilon > 0\) 最终 \(|a_n - b_n| < \varepsilon\)"， 两者就 \(\varepsilon/2\) 一调即通。

关键桥接定理

（equivR_eq'，关键）。 对任何柯西序列 \(a\)， \[ \mathrm{equivR}\bigl(\mathrm{LIM}\, a\bigr) \;=\; \mathrm{Real.mk}\bigl(a\bigr). \] （右侧的 \(\mathrm{Real.mk}\) 是 Mathlib 把柯西序列变成实数的算子。）

这是整个尾声章节的真正的心脏：它把第 5 章的"形式极限"翻译成 Mathlib 的标准极限。 这条引理一旦建立，把任何用柯西序列定义的运算（加、乘、序）翻译到 \(\mathbb{R}\) 一侧就只需要例行公事。

分两种情形。
  • 情形 1： 若 \(\mathrm{LIM}\, a = q\) 是有理数。则一边是 \((q:\mathbb{R})\)（由 equivR_ratCast）， 另一边由常值序列 \(q,q,q,\ldots\) 与 \(a\) 等价及 \(\mathrm{Real.mk}\) 的常数性可知也是 \((q:\mathbb{R})\)。
  • 情形 2： 若 \(\mathrm{LIM}\, a\) 不是有理数。 要证 \(\sup\{(q:\mathbb{R}) : q \in E_{\mathrm{LIM}\, a}\} = \mathrm{Real.mk}\, a\)。 分两步：
    • (上界) 任取 \(q \in \mathbb{Q}\) 使 \((q:\mathbf{R}_{\text{Cau}}) < \mathrm{LIM}\, a\)。 利用第 5 章的"序与 LIM 的相容性"，\(a_n\) 最终大于 \(q\)。 由 Mathlib 的 Real.mk_le 类引理，\((q:\mathbb{R}) \leq \mathrm{Real.mk}\, a\)。
    • (最小性) 假设 \(M < \mathrm{Real.mk}\, a\) 是上界。由有理稠密性取 \(q \in (M, \mathrm{Real.mk}\, a)\)。 类似地推出 \((q:\mathbf{R}_{\text{Cau}}) < \mathrm{LIM}\, a\)，故 \((q:\mathbb{R})\) 在我们考察的集合里， 从而 \(M < q\) 应被达到，矛盾。
📖 "形式极限 = 真正极限" 这件事并不平凡

第 5 章构造 \(\mathbf{R}_{\text{Cau}}\) 的方式是：给定柯西序列 \(a\)，定义一个新对象 \(\mathrm{LIM}\, a\)， 并通过等价类"硬塞"\(\mathbf{R}_{\text{Cau}}\) 的代数与序结构。 它是不是真的等于"\(a_n\) 的极限"？在第 5 章语境下这个问题没有意义，因为彼时还没有定义"极限"。

这条桥接引理回过头来证明：当我们最终建立了极限理论后，形式极限就是真正的极限。 这把一切努力封闭起来——我们没有做无用功，我们造出来的就是教科书里的那个 \(\mathbb{R}\)。

注：原 Lean 文件中 equivR_eq' 及它依赖的几个辅助引理 （cauchy_equiv_of_close、exists_equiv_above、exists_equiv_below） 仍带有 sorry，是留给读者完成的练习。它们都属于"原则上无难度，体力上需细致"的类型。
— 环结构与序结构的保持 —

有了 equivR_eq'，把代数运算翻译过去就成了固定套路。 对任何 \(x, y \in \mathbf{R}_{\text{Cau}}\)，可写成 \(x = \mathrm{LIM}\, a, y = \mathrm{LIM}\, b\)。 则 \(x + y = \mathrm{LIM}\,(a+b)\)，故 \[ \mathrm{equivR}(x+y) = \mathrm{Real.mk}(a+b) = \mathrm{Real.mk}(a) + \mathrm{Real.mk}(b) = \mathrm{equivR}(x) + \mathrm{equivR}(y). \] 乘法、序、负元、倒数同理。

（equivR_ordered_ring）。 \(\mathrm{equivR}\) 是： \[ \mathbf{R}_{\text{Cau}} \;\;\simeq^{+,\times,\leq}\;\; \mathbb{R}. \] 具体而言，\(\mathrm{equivR}\) 把 \(0 \mapsto 0\)、\(1 \mapsto 1\)、并保持加法、乘法、不等式 \(\leq\)。

保序环同构自动保持所有由 \(+, \times, 0, 1, \leq\) 派生的结构：减法、倒数、绝对值、上确界、下确界等等。 具体地，

\(\mathrm{equivR}(-x) = -\mathrm{equivR}(x),\qquad \mathrm{equivR}(x^{-1}) = (\mathrm{equivR}\,x)^{-1},\)
\(0 < x \;\Longleftrightarrow\; 0 < \mathrm{equivR}(x),\qquad 0 \leq x \;\Longleftrightarrow\; 0 \leq \mathrm{equivR}(x).\)
— 指数运算的保持 —

最后还要验证"指数运算"——第 5.6 节费力建立起来的那一系列方根和有理指数——也被同构保持。

自然数指数

（pow_of_equivR）。 对 \(x \in \mathbf{R}_{\text{Cau}}\)，\(n \in \mathbb{N}\)， \[ \mathrm{equivR}(x^n) \;=\; (\mathrm{equivR}\, x)^n. \]
对 \(n\) 归纳。\(n = 0\)：两边均为 \(1\)，由 \(\mathrm{equivR}(1) = 1\)。 归纳步骤：\(\mathrm{equivR}(x^{n+1}) = \mathrm{equivR}(x^n \cdot x) = \mathrm{equivR}(x^n) \cdot \mathrm{equivR}(x) = (\mathrm{equivR}\,x)^n \cdot \mathrm{equivR}(x) = (\mathrm{equivR}\,x)^{n+1}\)， 用了乘法的保持和归纳假设。

整数指数

（zpow_of_equivR）。 对 \(x \in \mathbf{R}_{\text{Cau}}, n \in \mathbb{Z}\)， \(\mathrm{equivR}(x^n) = (\mathrm{equivR}\, x)^n.\)
对负幂 \(n = -m\)，\(x^{-m} = (x^m)^{-1}\)。再用 \(\mathrm{equivR}\) 保持乘法逆和上一引理。

有理指数（与 Mathlib 的 rpow 对接）

（ratPow_of_equivR，核心兼容性）。 对 \(x \in \mathbf{R}_{\text{Cau}}, q \in \mathbb{Q}\)， \[ \mathrm{equivR}(x^q) \;=\; (\mathrm{equivR}\, x)^{(q:\mathbb{R})}. \] 左侧的 \(x^q\) 是第 5.6 节定义的（Real.ratPow）， 右侧的 \(\;\cdot^{(q:\mathbb{R})}\) 是 Mathlib 的（Real.rpow） 在有理数处的取值。
📖 这条引理为什么重要

从第 6 章开始，整本书都将使用 Mathlib 的 \(\mathbb{R}\) 及其 rpow。 但第 5.6 节的 ratPow 是另一套定义， 通过"\(b\) 次方根，再 \(a\) 次幂"来构造。 没有这条引理，第 5 章建立的所有有理指数性质都用不到后续章节中。

这条引理就是说：两套定义殊途同归。 数学上当然是显然的（毕竟正实数的有理指数本来就只有一种合理定义）， 但在形式化中需要通过同构 \(\mathrm{equivR}\) 把"两套"明确等同起来。

把 \(q\) 写成 \(q = a/b\)，其中 \(a \in \mathbb{Z}, b \in \mathbb{N}^{+}\)。则 \(x^q = (x^{1/b})^a\)（第 5.6 节定义）。两侧分别处理：
  • 方根的保持： 设 \(y := x^{1/b}\)，则 \(y \geq 0\) 且 \(y^b = x\)。应用 \(\mathrm{equivR}\)： \(\mathrm{equivR}(y) \geq 0\) 且 \((\mathrm{equivR}\, y)^b = \mathrm{equivR}(x)\)。 由 \(\mathbb{R}\) 中方根的唯一性，\(\mathrm{equivR}(y) = (\mathrm{equivR}\, x)^{1/b}\)。
  • 整数幂的保持： 已由 zpow_of_equivR 给出。
  • 合起来： \(\mathrm{equivR}(x^q) = \mathrm{equivR}((x^{1/b})^a) = (\mathrm{equivR}(x^{1/b}))^a = ((\mathrm{equivR}\,x)^{1/b})^a = (\mathrm{equivR}\,x)^{a/b} = (\mathrm{equivR}\,x)^{(q:\mathbb{R})}\)。
最后一步用了 Mathlib 中 rpow 在有理指数处与方根复合的定义性等式。
— 意义：为何要做这件事？—

有人可能会问：既然 Mathlib 已经有 \(\mathbb{R}\)，为什么 Tao 要"再造一遍"，然后又花一个尾声章节证明它和 Mathlib 的同构？ 答案分两个层面。

数学层面

教科书《分析学 I》的目的之一就是让读者从零构造实数，亲身体会"完备化"的全过程。 跳过这一构造，直接使用 \(\mathbb{R}\)，就丧失了第 5 章的教学价值。 但教学价值不等于实用价值——构造一旦完成，所有进一步的分析（极限、连续、微积分） 应该建立在一个统一的、有完整 API 的实数对象之上。 本尾声章节正是把"教学构造"和"实用对象"严格地连接起来。

形式化层面

更具体地说，从第 6 章开始，本 Lean 形式化项目将使用 Mathlib 的 \(\mathbb{R}\) 而非 Chapter5.Real。 这一过渡的合理性正是依赖于 equivR 是保结构同构。 有了它：

  • 第 5 章证明的任何关于 Chapter5.Real 的定理，都能通过 equivR 转移到 \(\mathbb{R}\) 上；
  • 反之，Mathlib 中关于 \(\mathbb{R}\) 的丰富 API（极限、拓扑、测度、积分），可以为后续章节直接所用；
  • 读者无需在两套实数之间反复切换——同构在某种意义上让 Chapter5.Real "退役"。
💬 关于"过度工程"的辩护

本尾声章节的代码量并不小，并且证明明显比第 5.1 至 5.6 节繁琐： 它要不断在两个不同（但同构）的世界之间翻译。 乍看之下这像是"过度工程"——既然两个 \(\mathbb{R}\) 是同构的，何必如此较真？

但形式化的灵魂正在于此。"两个对象同构"在数学中是结论， 在形式化中是需要被构造的同构。 没有显式的 equivR，Lean 不会自动认为 Chapter5.Real.add 和 Real.add 是"同一种"加法。 本章把这种"显然的同一性"翻译成 Lean 看得懂的语言，从而打开了通向后续章节的大门。

— 补充说明 —
📖 三种实数构造的视角

历史上构造实数的方法不止两种，常见的有：

  • 戴德金分割（Dedekind, 1872）：本章的"桥梁"。直接，但代数运算定义繁琐。
  • 柯西序列商（Cantor, Méray）：本书第 5 章。代数运算自然，但"等价类"略显抽象。
  • 无尽小数展开：直观，但小数运算（如 \(0.999\ldots = 1\)）需要小心处理。

更现代的方法还有"完备阿基米德有序域的范畴是连通的（任何两个对象间存在唯一同构）" 这个范畴论刻画。本尾声证明的事实是这一连通性的具体实例。

📖 形式化中的 Equiv 与 ≃+*o

Lean/Mathlib 中关于"同构"有多种类型：

  • ≃（Equiv）：仅集合双射，不要求保持任何结构；
  • ≃+：保加法的群同构；
  • ≃+*：保加法和乘法的环同构；
  • ≃+*o：保加法、乘法和序的有序环同构。

本章的 Real.equivR 是 ≃，而升级版 Real.equivR_ordered_ring 是 ≃+*o。 后者几乎"什么都保持"，因此可以从它派生出 map_inv、map_pos 等无穷多个推论。

💬 个人感悟：再见，Chapter5.Real

写到这里，第 5 章就真正落幕了。从 5.1 节的 \(\varepsilon\)-稳定序列， 到 5.6 节的有理指数，再到本尾声的同构桥，我们花了相当多的篇幅从零构造一个实数系， 然后又花一整章证明它与 Mathlib 的标准构造等价。

回过头看，整个旅程最值得学习的不是"哪种构造更优"， 而是"什么样的对象配得上叫实数"—— 它必须是一个完备阿基米德有序域，并且这个性质在同构的意义下是唯一的。 所有的构造（柯西序列、戴德金分割、无尽小数）都在追逐同一个不变量。

从下一章起，我们就把 Chapter5.Real 这个老朋友放下， 转而使用 Mathlib 的 ℝ。但有了本尾声章节的 equivR， 我们随时可以把第 5 章学到的一切翻译过去。这种"翻译自由"正是同构的礼物。

— 第 5 章尾声·完 —

本章证明了第 5 章的柯西序列实数 \(\mathbf{R}_{\text{Cau}}\) 与 Mathlib 的标准实数 \(\mathbb{R}\) 通过戴德金分割 ， 且这一同构保持所有的加法、乘法、序、有理嵌入、自然数/整数/有理指数。 从下一章开始，Chapter5.Real 退役，我们将直接使用 \(\mathbb{R}\)。
