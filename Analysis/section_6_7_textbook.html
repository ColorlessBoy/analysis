— 引言 —

我们终于回到第 5.6 节开始讨论的实数指数运算的话题。在那一节中我们对所有正实数 \(x\) 和有理数 \(q\) 定义了 \(x^q\)， 但尚未定义 \(\alpha\) 为实数时的 \(x^\alpha\)。我们现在利用极限来弥补这一缺憾 （类似于我们定义实数上其他标准运算的方式）。首先，我们需要一个引理。

6.7.1：指数运算的连续性

6.7.1（指数运算的连续性）。 设 \(x > 0\)，\(\alpha\) 是实数。设 \((q_n)_{n=1}^\infty\) 是收敛到 \(\alpha\) 的有理数序列。 则 \((x^{q_n})_{n=1}^\infty\) 也是收敛序列。 此外，若 \((q'_n)_{n=1}^\infty\) 是任何其他收敛到 \(\alpha\) 的有理数序列， 则 \((x^{q'_n})_{n=1}^\infty\) 与 \((x^{q_n})_{n=1}^\infty\) 有相同的极限： \[ \lim_{n\to\infty} x^{q_n} = \lim_{n\to\infty} x^{q'_n}. \]
证明.

分三种情形：\(x < 1\)、\(x = 1\)、\(x > 1\)。\(x = 1\) 的情形很简单 （因为对所有有理数 \(q\) 有 \(x^q = 1\)）。我们只处理 \(x > 1\) 的情形， \(x < 1\) 的情形非常类似，留给读者。

第一步：证明 \((x^{q_n})_{n=1}^\infty\) 收敛。 由命题 6.4.18，只需证明 \((x^{q_n})_{n=1}^\infty\) 是 Cauchy 序列。 为此需估计 \(x^{q_n}\) 与 \(x^{q_m}\) 的距离；暂设 \(q_n \geq q_m\)，则 \(x^{q_n} \geq x^{q_m}\)（因为 \(x > 1\)）。我们有 \[ d(x^{q_n}, x^{q_m}) = x^{q_n} - x^{q_m} = x^{q_m}(x^{q_n - q_m} - 1). \] 由于 \((q_n)_{n=1}^\infty\) 收敛，它有某个上界 \(M\)；因为 \(x > 1\)，有 \(x^{q_m} \leq x^M\)。故 \[ d(x^{q_n}, x^{q_m}) = |x^{q_n} - x^{q_m}| \leq x^M (x^{q_n - q_m} - 1). \]

令 \(\varepsilon > 0\)。由引理 6.5.3 知序列 \((x^{1/k})_{k=1}^\infty\) 最终 \(\varepsilon x^{-M}\)-接近 1， 故存在 \(K \geq 1\) 使得 \(|x^{1/K} - 1| \leq \varepsilon x^{-M}\)。

由于 \((q_n)_{n=1}^\infty\) 收敛，它是 Cauchy 序列，故存在 \(N \geq 1\) 使得对所有 \(n, m \geq N\)， \(q_n\) 与 \(q_m\) 是 \(1/K\)-接近的。于是 \[ d(x^{q_n}, x^{q_m}) \leq x^M (x^{q_n - q_m} - 1) \leq x^M (x^{1/K} - 1) \leq x^M \varepsilon x^{-M} = \varepsilon \] 对所有 \(n, m \geq N\) 且 \(q_n \geq q_m\) 成立。由对称性，\(q_n \leq q_m\) 时也有同样估计。 因此 \((x^{q_n})_{n=N}^\infty\) 是 \(\varepsilon\)-稳定的，从而 \((x^{q_n})_{n=1}^\infty\) 对每个 \(\varepsilon > 0\) 都是最终 \(\varepsilon\)-稳定的， 因而是 Cauchy 序列。这就证明了收敛性。

第二步：证明极限与序列选取无关。 只需证明 \(\lim_{n\to\infty} x^{q_n - q'_n} = 1\)， 则由极限定律（\(x^{q_n} = x^{q_n - q'_n} x^{q'_n}\)）即得结论。 令 \(r_n := q_n - q'_n\)；由极限定律知 \((r_n)_{n=1}^\infty\) 收敛到 0。 需证对每个 \(\varepsilon > 0\)，\((x^{r_n})_{n=1}^\infty\) 最终 \(\varepsilon\)-接近 1。 由引理 6.5.3 知 \((x^{1/k})_{k=1}^\infty\) 最终 \(\varepsilon\)-接近 1， 且 \(\lim_{k\to\infty} x^{-1/k} = 1\) 也成立，故 \((x^{-1/k})_{k=1}^\infty\) 也最终 \(\varepsilon\)-接近 1。 从而可找到 \(K\) 使 \(x^{1/K}\) 和 \(x^{-1/K}\) 都 \(\varepsilon\)-接近 1。 而 \((r_n)_{n=1}^\infty\) 收敛到 0，故最终 \(1/K\)-接近 0， 即最终 \(-1/K \leq r_n \leq 1/K\)，从而 \(x^{-1/K} \leq x^{r_n} \leq x^{1/K}\)。 特别地，\(x^{r_n}\) 也最终 \(\varepsilon\)-接近 1（见命题 4.3.7(f)），得证。

∎

6.7.2：实指数运算

我们现在可以做如下定义。

6.7.2（实数指数的指数运算）。 设 \(x > 0\) 为实数，\(\alpha\) 为实数。定义量 \(x^\alpha\) 为 \[ x^\alpha = \lim_{n\to\infty} x^{q_n}, \] 其中 \((q_n)_{n=1}^\infty\) 是收敛到 \(\alpha\) 的任意有理数序列。

让我们验证这个定义是良定义的。首先，给定任意实数 \(\alpha\)， 由实数的定义（和命题 6.1.15），我们总至少有一个收敛到 \(\alpha\) 的有理数序列 \((q_n)_{n=1}^\infty\)。 其次，给定任何这样的序列，极限 \(\lim_{n\to\infty} x^{q_n}\) 由引理 6.7.1 存在。 最后，虽然序列 \((q_n)_{n=1}^\infty\) 可以有多种选择，但由引理 6.7.1 它们都给出相同的极限。 因此这个定义是良定义的。

如果 \(\alpha\) 不仅是实数还是有理数，即 \(\alpha = q\) 为某有理数， 则此定义原则上可能与第 5.6 节中早先的指数定义不一致。但此时 \(\alpha\) 显然是常数序列 \((q)_{n=1}^\infty\) 的极限， 故由定义 \(x^\alpha = \lim_{n\to\infty} x^q = x^q\)。因此新的指数定义与旧定义一致。

补充说明：Lean 中的定义

由于第 5 章的实数已被弃用，改用 Mathlib 的实数，而 Mathlib 的实指数运算不需要先经过有理指数， Lean 形式化采用了一种折中方案：先接受 Mathlib 的有理指数运算（及其全部 API）， 再用它定义实指数运算，并在本章尾声中证明它与 Mathlib 运算一致。

/-- 每个实数都是某有理数序列的极限 -/ theorem Real.eq_lim_of_rat (α:ℝ) : ∃ q: ℕ → ℚ, ((fun n ↦ (q n:ℝ)):Sequence).TendsTo α /-- Definition 6.7.2 (Exponentiation to a real exponent) -/ noncomputable abbrev Real.rpow (x:ℝ) (α:ℝ) :ℝ := lim ((fun n ↦ x^((eq_lim_of_rat α).choose n:ℝ)):Sequence)

关键辅助引理保证良定义性——不同的有理逼近序列给出相同极限：

lemma Real.rpow_eq_lim_ratPow {x α:ℝ} (hx: x > 0) {q: ℕ → ℚ} (hq: ((fun n ↦ (q n:ℝ)):Sequence).TendsTo α) : rpow x α = lim ((fun n ↦ x^(q n:ℝ)):Sequence)
补充说明：与 Mathlib 指数运算的同一性

本节定义的 Real.rpow 最终被证明等于 Mathlib 的标准指数运算 x ^ α：

/-- Bridge lemma: Chapter6's sequence convergence ↔ Mathlib's filter convergence -/ lemma tendsto_iff_tendsto_nhds (a : ℕ → ℝ) (L : ℝ) : ((a : Sequence).TendsTo L) ↔ atTop.Tendsto a (nhds L) /-- Identification with Mathlib exponentiation -/ theorem Real.rpow_eq_rpow {x : ℝ} (hx : x > 0) (α : ℝ) : rpow x α = x ^ α

证明利用 Mathlib 的连续性定理 Real.continuousAt_const_rpow： 由于 \(t \mapsto x^t\) 在 \(\alpha\) 处连续，而 \(q_n \to \alpha\)（在 Mathlib 的滤语言义下）， 故 \(x^{q_n} \to x^\alpha\)。同时由定义 \(x^{q_n} \to \texttt{rpow}\, x\, \alpha\)。 由滤极限的唯一性（tendsto_nhds_unique），二者相等。

6.7.3：实指数的运算法则

6.7.3。 引理 5.6.9 中对有理数 \(q\) 和 \(r\) 成立的所有结果，对实数 \(q\) 和 \(r\) 继续成立。即对 \(x, y > 0\) 和实数 \(q, r\)：
  • (a) \(x^q \geq 0\)；进一步，若 \(x \neq 0\) 则 \(x^q > 0\)。
  • (b) \(x^{q+r} = x^q \cdot x^r\) 且 \((x^q)^r = x^{qr}\)；
  • (c) \(x^{-q} = 1 / x^q\)；
  • (d) 当 \(q > 0\) 时，\(x > y \iff x^q > y^q\)；
  • (e) 当 \(x > 1\) 时，\(x^q > x^r \iff q > r\)；当 \(0 < x < 1\) 时，\(x^q > x^r \iff q < r\)；
  • (f) \((xy)^q = x^q \cdot y^q\)。
证明.

我们以恒等式 \(x^{q+r} = x^q x^r\)（即引理 5.6.9(b) 的第一部分）为例来演示； 其他部分类似，留给习题 6.7.1。核心思想是从有理数情形的引理 5.6.9 出发，取极限得到实数情形的对应结果。

设 \(q\) 和 \(r\) 是实数。由实数的定义（和命题 6.1.15），可以写出 \(q = \lim_{n\to\infty} q_n\) 和 \(r = \lim_{n\to\infty} r_n\)， 其中 \((q_n)_{n=1}^\infty\) 和 \((r_n)_{n=1}^\infty\) 是有理数序列。 由极限定律，\(q + r\) 是 \((q_n + r_n)_{n=1}^\infty\) 的极限。 由实指数的定义： \[ x^{q+r} = \lim_{n\to\infty} x^{q_n + r_n}; \quad x^q = \lim_{n\to\infty} x^{q_n}; \quad x^r = \lim_{n\to\infty} x^{r_n}. \] 但由引理 5.6.9(b)（应用于有理指数）有 \(x^{q_n + r_n} = x^{q_n} x^{r_n}\)。 再由极限定律得 \(x^{q+r} = x^q x^r\)，证毕。

∎

个人感悟："取极限"作为定义的方法论

第 6.7 节展示了一种极其重要的数学方法论：用极限来扩展运算的定义域。 我们已经有有理指数 \(x^q\)，现在要定义实指数 \(x^\alpha\)。 方法是：取一列有理数 \(q_n \to \alpha\)，定义 \(x^\alpha := \lim x^{q_n}\)。

这个方法之所以可行，需要两个条件：(1) 存在性——极限存在（引理 6.7.1 第一部分， 依赖 Cauchy 完备性 6.4.18 和引理 6.5.3）；(2) 唯一性——不同的逼近序列给出相同的极限 （引理 6.7.1 第二部分）。这两个条件正是"良定义"的全部要求。

在 Lean 形式化中，这套方法遇到了一个有趣的张力：Mathlib 的实指数 x ^ α 已经存在， 而且不需要先经过有理指数。教材的"从有理取极限"路径在 Mathlib 中并非标准做法。 形式化项目采取了折中：先定义教材版本的 Real.rpow，再证明它等于 Mathlib 的 ^。 这种"先忠实翻译，再桥接标准库"的策略，既保留了教材的教学价值，又获得了 Mathlib 的全部 API。
