— 引言 —

第 4.3 节定义了当 \(x\) 为、\(n\) 为自然数或整数时的指数运算 \(x^n\)。 现在我们已经构造了实数并建立了上确界性质，我们可以将这些指数运算推广到实数上， 并进一步定义 \(n\) 次方根和有理指数。

本节称为"实指数运算（上）"，因为我们只处理有理指数。 真正的实指数（即 \(x^y\) 其中 \(y\) 为任意实数）需要等到第 6.7 节， 在建立了极限理论之后才能定义。

5.6.1：自然数指数

5.6.1（实数的自然数指数）。 设 \(x\) 为实数。
  • 定义 \(x^0 := 1\)；
  • 递归地，若 \(x^n\) 已定义，则 \(x^{n+1} := x^n \times x\)。

5.6.2：整数指数

5.6.2（实数的整数指数）。 设 \(x\) 为非零实数。对负整数 \(-n\)（\(n > 0\)），定义 \[ x^{-n} := \frac{1}{x^n}. \]

5.6.3：指数定律对实数仍然成立

5.6.3。 命题 4.3.10 和 4.3.12 中的所有指数性质，当 \(x\) 和 \(y\) 为实数而非有理数时仍然成立。
元证明： 命题 4.3.10 和 4.3.12 的证明仅依赖于有理数的代数定律和序定律 （命题 4.2.4 和 4.2.9）。而由命题 5.3.11、5.4.7 以及恒等式 \(xx^{-1} = x^{-1}x = 1\)， 这些定律对实数也成立。因此，我们可以直接修改原证明使其适用于实数。
📖 补充说明："元证明"的含义

这不是一个严格意义上的数学证明，而是一个关于证明本身的论证 （因此称为"元证明"——meta-proof）。它依赖于这样一个事实： 有理指数定律的证明只使用了对域和序都成立的公理，而这些公理对实数同样成立。 因此整个证明可以"原样照搬"。

在 Lean 中，这并不是直接复制粘贴——需要额外的工作来将有理数的指数定律 通过 Real.ratCast 提升到实数上（见 Section_5_6.lean 中 Real.pow_of_coe 等引理）。但从数学概念上讲，确实没有任何新的困难。

5.6.4：\(n\) 次方根

5.6.4（\(n\) 次方根）。 设 \(x \geq 0\) 为非负实数，\(n \geq 1\) 为正整数。定义 \(x\) 的 \(n\) 次方根为 \[ x^{1/n} := \sup\{y \in \mathbb{R} : y \geq 0 \text{ 且 } y^n \leq x\}. \] 当 \(n = 2\) 时，也记 \(\sqrt{x} := x^{1/2}\)。 负数的 \(n\) 次方根在本节中不予定义（留待复数理论中处理）。
注。 这个定义与命题 5.5.12 中 \(\sqrt{2}\) 的构造一致： \(\sqrt{2}\) 就是集合 \(\{y \geq 0 : y^2 \leq 2\}\) 的上确界。

5.6.5：方根的存在性

5.6.5（\(n\) 次方根的存在性）。 设 \(x \geq 0\)，\(n \geq 1\)。则集合 \(E := \{y \in \mathbb{R} : y \geq 0 \text{ 且 } y^n \leq x\}\) 非空且有上界。特别地，\(x^{1/n}\) 是一个实数。
\(E\) 显然非空（\(0 \in E\)）。下设上界：
  • 若 \(x \leq 1\)，则 1 是上界（若 \(y > 1\)，则 \(y^n > 1 \geq x\)，矛盾）；
  • 若 \(x > 1\)，则 \(x\) 本身是上界（若 \(y > x > 1\)，则 \(y^n > x\)，矛盾）。
因此由定理 5.5.9，\(E\) 有最小上界，即 \(x^{1/n}\)。

5.6.6：方根的性质

5.6.6。 设 \(x, y \geq 0\)，\(n, m \geq 1\)。则：
  1. 若 \(y = x^{1/n}\)，则 \(y^n = x\)；
  2. 反之，若 \(y^n = x\)，则 \(y = x^{1/n}\)；
  3. \(x^{1/n} \geq 0\)，且 \(x^{1/n} > 0\) 当且仅当 \(x > 0\)；
  4. \(x > y\) 当且仅当 \(x^{1/n} > y^{1/n}\)；
  5. 若 \(x > 1\)，则 \(x^{1/k}\) 关于 \(k\) 递减；若 \(0 < x < 1\)，则递增；若 \(x = 1\)，则 \(x^{1/k} = 1\)；
  6. \((xy)^{1/n} = x^{1/n} y^{1/n}\)；
  7. \((x^{1/n})^{1/m} = x^{1/(nm)}\)。
见习题 5.6.1。
📖 补充说明：(a) 和 (b) 的证明思路

部分 (a) 和 (b) 说明 \(x^{1/n}\) 确实是 \(y^n = x\) 的唯一非负解。

(a) 的证明思路： 假设 \(y = x^{1/n}\) 但 \(y^n < x\)。 则可以找到 \(\varepsilon > 0\) 使得 \((y + \varepsilon)^n < x\)（利用类似命题 5.5.12 的放缩）， 从而 \(y + \varepsilon \in E\)，与 \(y\) 是上确界矛盾。同理可证 \(y^n > x\) 也导致矛盾。

(b) 的证明思路： 若 \(y^n = x\)，则 \(y\) 显然在 \(E\) 中且是 \(E\) 的上界 （因为若 \(z > y\)，则 \(z^n > y^n = x\)，故 \(z \notin E\)）。因此 \(y = \sup(E) = x^{1/n}\)。

5.6.7：有理指数

5.6.7（有理指数）。 设 \(x > 0\) 为正实数，\(q\) 为有理数。将 \(q\) 写为 \(q = a/b\)（其中 \(a \in \mathbb{Z}\)，\(b \in \mathbb{N}^+\)）， 定义 \[ x^q := (x^{1/b})^a. \]
注。 由于 \(q\) 可以写成不同的分数形式（如 \(1/2 = 2/4 = 3/6\)）， 我们需要验证这个定义不依赖于表示方式——这就是引理 5.6.8 的内容。

5.6.8：良定性

5.6.8。 设 \(a, a'\) 为整数，\(b, b'\) 为正整数，且 \(a/b = a'/b'\)。 设 \(x > 0\)。则 \((x^{1/b'})^{a'} = (x^{1/b})^a\)。 因此 \(x^q\) 的定义与 \(q\) 的表示无关。
分三种情况：\(a = 0\)、\(a > 0\)、\(a < 0\)。
  • 若 \(a = 0\)，则 \(a' = 0\)，两边均为 1；
  • 若 \(a > 0\)，则 \(a' > 0\)，且 \(ab' = ba'\)。 令 \(y := x^{1/(ab')}\)。由引理 5.6.6(g)，\(y = (x^{1/b'})^{1/a} = (x^{1/b})^{1/a'}\)。 由引理 5.6.6(a)，\(y^a = x^{1/b'}\)，\(y^{a'} = x^{1/b}\)。 因此 \((x^{1/b'})^{a'} = (y^a)^{a'} = y^{aa'} = (y^{a'})^a = (x^{1/b})^a\)；
  • 若 \(a < 0\)，则 \((-a)/b = (-a')/b'\)，由 \(a > 0\) 的情况得 \((x^{1/b'})^{-a'} = (x^{1/b})^{-a}\)，取倒数即得。

5.6.9：有理指数的基本性质

5.6.9。 设 \(x, y > 0\)，\(q, r\) 为有理数。则：
  1. \(x^q > 0\)；
  2. \(x^{q+r} = x^q x^r\)，且 \((x^q)^r = x^{qr}\)；
  3. \(x^{-q} = 1 / x^q\)；
  4. 若 \(q > 0\)，则 \(x > y\) 当且仅当 \(x^q > y^q\)；
  5. 若 \(x > 1\)，则 \(x^q > x^r\) 当且仅当 \(q > r\)；若 \(x < 1\)，则 \(x^q > x^r\) 当且仅当 \(q < r\)；
  6. \((xy)^q = x^q y^q\)。
见习题 5.6.2。
📖 补充说明：指向实指数的桥梁

引理 5.6.9 完成了有理指数的全部性质。但要定义实指数 \(x^y\) （其中 \(y\) 为任意实数），我们需要极限理论： 对任意实数 \(y\)，取一列有理数 \(q_n \to y\)，定义 \[ x^y := \lim_{n\to\infty} x^{q_n}. \] 这个极限的存在性需要证明（利用引理 5.6.9 的性质），这正是第 6.7 节的内容。

— 第 5.6 节总结 —

第 5.6 节将指数运算从有理数推广到了实数，涵盖三个层次：

  1. 自然数/整数指数： 通过递归定义和取倒数直接推广（定义 5.6.1–5.6.2）；
  2. \(n\) 次方根： 利用上确界性质构造（定义 5.6.4），证明其基本性质（引理 5.6.5–5.6.6）；
  3. 有理指数： 通过方根和整数指数的组合定义（定义 5.6.7），验证良定性（引理 5.6.8）和代数性质（引理 5.6.9）。

至此，第 5 章（实数的构造）的主要内容全部完成。 在后续章节中，我们将把实数视为理所当然的数系，并发展极限、连续性、微分和积分的理论。

💬 个人感悟：实数构造的完成

从第 5.1 节的 Cauchy 序列定义开始，到第 5.6 节的有理指数运算为止， 我们完成了从有理数到实数的完整构造。这是分析学中最基本的一步—— 从此之后，我们可以"忘记"实数是用 Cauchy 序列构造的， 而把它们当作具有完备性的有序域来使用。

回看整个第 5 章，核心的技术贡献只有两个： Cauchy 序列（用于构造）和上确界性质（用于完备性）。 其余的一切——加法、乘法、序、倒数、方根——都只是在这两个概念上的自然展开。 正如 Tao 所说，数学的进步往往不是靠引入全新的概念， 而是靠把已有的概念用到极致。

📖 补充说明：Lean 中的方根定义

Real.root 通过上确界定义：

noncomputable abbrev Real.root (x:Real) (n:ℕ) : Real := sSup { y:Real | y ≥ 0 ∧ y^n ≤ x } noncomputable abbrev Real.sqrt (x:Real) := x.root 2

由于 Real 已注册为 ConditionallyCompleteLattice， 可以直接使用 sSup 而无需手动调用 LUB_exist。

📖 补充说明：有理指数在 Lean 中的实现

Real.ratPow 的定义：

noncomputable abbrev Real.ratPow (x:Real) (q:ℚ) : Real := (x.root q.den)^(q.num) noncomputable instance Real.instRatPow : Pow Real ℚ where pow x q := x.ratPow q

q.den 是有理数的分母，q.num 是分子。 良定性（引理 5.6.8）由 Real.pow_root_eq_pow_root 保证。
