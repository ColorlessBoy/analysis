Section 5.4: Ordering the Reals

本节的目标是在由柯西序列构造的实数上建立全序结构，并证明该序与加法、乘法兼容， 最终得到 LinearOrder Real 和 IsStrictOrderedRing Real 两个核心实例。 这使得实数具备了后续分析（极限、连续性、积分）所需的全部序论基础设施。

一、基础定义：正/负远离零的序列

定义 Definition 5.4.1 — BoundedAwayPos / BoundedAwayNeg

BoundedAwayPos a：存在 c > 0 使得 ∀ n, a(n) ≥ c

BoundedAwayNeg a：存在 c > 0 使得 ∀ n, a(n) ≤ −c

作用：这是定义实数"正"和"负"的根基。一个实数为正，当且仅当它可以用一个 BoundedAwayPos 的柯西序列来表示。 与单纯的"最终大于零"不同，bounded away 保证了与零有确定的距离，这使得性质在等价类下保持不变。

辅助 BoundedAwayZero.boundedAwayPos / boundedAwayNeg

正远离零 ⟹ 远离零；负远离零 ⟹ 远离零

作用：将有符号的 bounded away 退化为无符号版本，供 Section 5.3 中的倒数构造使用（倒数只需要远离零，不关心符号）。

辅助 not_boundedAwayPos_boundedAwayNeg

一个序列不可能同时 BoundedAwayPos 又 BoundedAwayNeg

作用：保证实数的正/负定义是互斥的，为三歧性做准备。

二、实数的正/负性质 (Proposition 5.4.4 / Exercise 5.4.1)

核心 Real.trichotomous — 三歧性

∀ x : Real, x = 0 ∨ x.IsPos ∨ x.IsNeg

作用：实数的序论基石。每个实数恰好属于正/负/零三者之一。这是后面建立 LinearOrder 的关键前提。

排除 Real.not_zero_pos / not_zero_neg / not_pos_neg

零不为正、零不为负、正不为负——三歧性中三者互斥。

作用：保证正/负/零划分是一个 partition，使得 if-then-else 分支推理合法。

对称 Real.neg_iff_pos_of_neg

x.IsNeg ↔ (−x).IsPos

作用：将负数的性质归约为正数的对称版本，避免重复证明。后续许多定理只需证正数情形，负数情形通过取反即得。

封闭性 Real.pos_add / Real.pos_mul

正数 + 正数 = 正数；正数 × 正数 = 正数

作用：正实数在加法和乘法下封闭。这是建立 IsStrictOrderedRing 实例的两个核心引理——直接对应"加法保序"和"正数乘法保序"。

嵌入 Real.pos_of_coe / Real.neg_of_coe

(q : Real).IsPos ↔ q > 0（有理数嵌入后保持正负性）

作用：确保有理数到实数的嵌入与序兼容。后续证明中经常需要"1 > 0"之类的事实，依赖此引理。

三、绝对值与序的定义 (Definition 5.4.5–5.4.6)

定义 Real.abs

|x| = x（若 x > 0），−x（若 x < 0），0（若 x = 0）

作用：为实数定义绝对值函数，用于度量和距离。后续通过 abs_eq_abs 证明它与 LinearOrder 自带的 |·| 一致。

定义 Real.instLT / Real.instLE

x < y ⟺ (x − y).IsNeg；x ≤ y ⟺ x < y ∨ x = y

作用：在实数上定义严格小于和小于等于。这是整个序结构的起点——后续所有不等式推理都建立在此定义之上。

四、序的基本性质 (Proposition 5.4.7 / Exercise 5.4.2)

结构 Real.trichotomous' — x > y ∨ x < y ∨ x = y

作用：序的三歧性。这是 LinearOrder 所要求的 le_total 的强化形式。

结构 Real.antisymm — x < y ↔ y > x

作用：反对称性。lt 和 gt 只是同一关系的两面，简化证明中方向切换。

结构 Real.lt_trans — 传递性

x < y ∧ y < z ⟹ x < z

作用：序的传递性，LinearOrder 的必要条件。几乎所有不等式链式推理都依赖此性质。

兼容 Real.add_lt_add_right — 加法保序

x < y ⟹ x + z < y + z

作用：有序加群（ordered additive group）的核心公理。允许在不等式两边同加一个数。IsStrictOrderedRing 的必要条件之一。

兼容 Real.mul_lt_mul_right — 正数乘法保序

x < y ∧ z > 0 ⟹ x·z < y·z

作用：有序环的核心公理。允许在不等式两边同乘正数。IsStrictOrderedRing 的必要条件之一。分析中大量估计都用到"同乘正 ε"。

五、全序实例 (LinearOrder Real)

实例 Real.instLinearOrder

组装 le_refl、le_trans、le_antisymm、le_total 得到 LinearOrder Real

作用：这是本节最重要的产出之一。有了 LinearOrder，实数自动获得：

  • min、max 函数
  • Decidable (a ≤ b)（通过 Classical）
  • Mathlib 中所有关于 LinearOrder 的引理（如 lt_or_ge、le_max_left 等）
  • 可以对实数使用 if/cases 比较

一致性 Real.abs_eq_abs

Mathlib 的 |x|（来自 LinearOrder 的 max x (−x)）等于我们定义的 abs x

作用：保证两种绝对值定义等价，后续可以自由互换使用。

六、正数的倒数 (Proposition 5.4.8)

保号 Real.inv_of_pos / div_of_pos

x > 0 ⟹ x⁻¹ > 0；x > 0 ∧ y > 0 ⟹ x/y > 0

作用：正数取倒数或做除法后仍为正。这是分析中"除以正数不变号"推理的基础。

反序 Real.inv_of_gt

0 < y < x ⟹ x⁻¹ < y⁻¹

作用：正数取倒数反转大小关系。经典性质，在 ε-δ 证明中频繁使用（如"取 N 使得 1/N < ε"）。

七、严格有序环实例 (IsStrictOrderedRing Real)

实例 Real.instIsStrictOrderedRing

组装 add_le_add_left/right、mul_lt_mul_of_pos_left/right、zero_le_one 等

作用：本节另一核心产出。有了 IsStrictOrderedRing，实数自动获得：

  • Mathlib 中所有有序环引理（如 mul_nonneg、sq_nonneg、abs_mul 等）
  • 可以使用 positivity、linarith、nlinarith 等 tactic
  • 数值分析中标准的不等式操作全部可用

八、极限与序的交互 (Proposition 5.4.9–5.4.10)

闭性 Real.LIM_of_nonneg — 非负实数的闭性

若 ∀ n, a(n) ≥ 0 且 a 是柯西序列，则 LIM a ≥ 0

作用：非负序列的极限仍非负。这是实分析中"不等式可以取极限"的基础版本。注意严格不等式不一定保持（见 Remark 5.4.11）。

保序 Real.LIM_mono — 极限保持弱序

若 ∀ n, a(n) ≤ b(n)，则 LIM a ≤ LIM b

作用：逐项不等式在取极限后仍成立（弱形式）。这是比较判别法、夹逼定理等的基础工具。

反例 Real.LIM_mono_fail sorry

存在 a(n) > b(n) 对所有 n 成立，但 LIM a ≯ LIM b

作用：说明严格不等式在取极限后可能退化为等号。经典反例：1 + 1/n > 1 − 1/n，但极限都是 1。提醒我们只能用 ≤ 而非 < 穿越极限。

九、有界性与阿基米德性质 (Proposition 5.4.12–5.4.13)

有界 Real.exists_rat_le_and_nat_gt — 用有理数/自然数夹逼

x > 0 ⟹ ∃ q ∈ ℚ 使 q ≤ x，且 ∃ N ∈ ℕ 使 x < N

作用：正实数可以从下方用正有理数逼近，从上方用自然数控制。这是阿基米德性质的直接前驱，也是实数稠密性论证的起点。

阿基米德 Real.le_mul — 阿基米德性质

∀ ε > 0, ∀ x, ∃ M > 0 使得 M·ε > x

作用：阿基米德性质的核心表述——没有"无穷大"实数。等价于 ℕ 在 ℝ 中无上界。 这是 ε-N 论证的基础：给定 ε > 0，总能找到足够大的 N 使得 N·ε 超过任何给定实数。

十、练习题 (Exercises 5.4.3–5.4.9)

练习 Real.floor_exist — 整数部分存在唯一性 sorry

∀ x, ∃! n ∈ ℤ, n ≤ x < n + 1

作用：实数的 floor 函数存在且唯一。这是定义取整函数 ⌊x⌋ 的理论依据，在数论和分析中广泛使用。

练习 Real.exist_inv_nat_le sorry

x > 0 ⟹ ∃ N > 0 使 1/N < x

作用：阿基米德性质的等价形式。在 ε-δ 证明中极常用：给定任意小的正实数 ε，总能找到自然数 N 使得 1/N < ε。

练习 Real.rat_between — 有理数稠密性 done

x < y ⟹ ∃ q ∈ ℚ, x < q < y

作用：有理数在实数中稠密（ℚ 在 ℝ 中稠密）。这是实分析的基本拓扑事实，在逼近论中处处用到。

练习 Real.dist_lt_iff / dist_le_iff — 距离不等式等价形式 sorry

|x − y| < ε ⟺ y − ε < x < y + ε

作用：将绝对值不等式拆解为双边不等式。这是 ε-邻域和开球定义的核心等价形式，在极限证明中每步都会用到。

练习 Real.le_add_eps_iff / dist_le_eps_iff sorry

(∀ ε > 0, x ≤ y + ε) ↔ x ≤ y；(∀ ε > 0, |x−y| ≤ ε) ↔ x = y

作用："任意小扰动下成立的不等式在极限下也成立"。这是分析中常见的论证模式——通过对所有 ε > 0 成立来推出极限处的结论。

练习 Real.LIM_of_le / LIM_of_ge sorry

若 ∀ n, a(n) ≤ x 则 LIM a ≤ x；若 ∀ n, a(n) ≥ x 则 LIM a ≥ x

作用：常数界在取极限后保持。是 LIM_mono 的推论形式，用于证明极限落在闭区间内。

练习 Real.max/min 系列 sorry

neg_max、neg_min、max_comm、max_self、max_add、max_mul、min 对应版本、inv_max、inv_min

作用：max/min 与代数运算的交互律。这些是 lattice 结构的基本性质，在优化问题、截断函数、正部/负部分解等场景中不可或缺。

嵌入 Real.ratCast_ordered_hom sorry

ℚ →+*o ℝ 是保序环同态

作用：有理数到实数的嵌入不仅保持加法和乘法（环同态），还保持序关系。这使得有理数中的不等式可以直接提升到实数使用。

总结：本节的逻辑主线

步骤内容产出
1定义 BoundedAwayPos/Neg序列层面的"正/负"概念
2定义 Real.IsPos/IsNeg，证明三歧性实数层面的正/负/零划分
3定义 < 和 ≤实数上的序关系
4证明序的公理性质LinearOrder Real
5证明序与加法、乘法兼容IsStrictOrderedRing Real
6极限保序、阿基米德性质分析所需的序列-序交互工具

一句话总结：本节将"柯西序列构造的实数"从一个代数对象（环）升级为一个有序代数对象（全序域）， 并建立了极限操作与序结构之间的基本桥梁。
